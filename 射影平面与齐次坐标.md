# 射影平面 $\mathbb{RP}^2$ 与齐次坐标

> [!abstract] 一句话摘要
> 射影平面之所以定义成 $\mathbb{RP}^2 := (\mathbb{R}^3 \setminus \{0\})\,/\sim$，
> 是因为这个定义**同时**做对了三件事：
> 1. 给"无穷远点"一个**真正的坐标**（不再是哲学口号）；
> 2. 让"平行线相交"成为**定理**（而非例外）；
> 3. 让所有几何变换变成**线性矩阵乘法**（图形学的福音）。

---

## 0. 预备：我们想解决什么问题？

欧几里得平面 $\mathbb{R}^2$ 有两个让人难受的"例外情况"：

| 例外 | 问题 |
| ---- | ---- |
| 两条**平行**直线 | 没有交点（其它直线都有交点） |
| 透视投影下"消失点" | 现实中铁轨在远处汇聚，但 $\mathbb{R}^2$ 描述不了这个"会合点" |

> [!question] 核心动机
> 能不能给 $\mathbb{R}^2$ "补"上一些点，使得：
> **"任意两条不同直线都恰好交于一点"** 成为无例外的定理？

这些"补上去的点"，就是后面要严格构造的**无穷远点**。

---

## 1. 朴素思路：直接"加一圈无穷远点"

先尝试一个最朴素的想法：

> 给 $\mathbb{R}^2$ 加一组新点，每个**方向**对应一个无穷远点。

### 1.1 为什么是"方向"对应一个点，而不是"每条平行线"对应一个点？

考虑斜率为 $k$ 的所有平行线：
$$y = kx + b_1,\quad y = kx + b_2,\quad y = kx + b_3,\ \ldots$$

它们彼此平行，**两两不相交**。如果想让它们"在无穷远处相交于同一点"，那这个点只能由**共同的方向 $k$** 决定，与截距 $b$ 无关。

> [!tip] 关键观察
> **方向 = 平行线族的不变量**
> → 一个无穷远点 ↔ 一个方向 ↔ 一族平行线

### 1.2 朴素方案的困难

朴素方案是把所有方向集合 $S^1 / \{\pm 1\}$ 强行塞回平面，但这样：

- 拓扑结构不清晰
- "无穷远点"和"普通点"看起来本质不同
- 任何变换公式里都要 if-else 区分两类点 → 工程灾难

> [!info] 怎么读懂 S¹ / {±1} 这个记号
>
> 这是数学家描述"所有方向"的标准记号，可以拆成两步理解：
>
> **Step 1：$S^1$ 是什么？**
> $S^1$ 表示 2 维平面中的**单位圆**：$\{(x,y) \mid x^2+y^2=1\}$。
> 圆上每个点对应一个**单位向量**，也就是平面上的一个**方向**。
> （比如 $(1,0)$ 是水平向右，$(0,1)$ 是竖直向上。）
>
> **Step 2：$/\{\pm 1\}$ 是什么？**
> 斜杠 "$/$" 在数学里读作 "模" 或 "商"，意思是**把某些点视为同一个**。
> $\{\pm 1\}$ 的作用规则是：把单位向量 $\vec v$ 和它的反向 $-\vec v$ **当成同一个东西**。
>
> **为什么要这样做？**
> 因为"方向"不分正反 —— 一条铁轨"向东"和"向西"的灭点是同一个无穷远点。
> 所以 $(1, 0)$ 和 $(-1, 0)$ 必须视为同一方向。
>
> **程序员视角的伪代码**
> ```python
> # S^1 / {±1} 就是这样一个集合
> directions = set()
> for theta in [0, 2*pi):
>     v = (cos(theta), sin(theta))           # S^1 上的一点
>     # 把 v 和 -v 视为同一个 → 只保留 theta ∈ [0, π)
>     if theta < pi:
>         directions.add(v)
> # 最终 directions ≅ 半圆 ≅ 所有"方向"
> ```
>
> 一句话：**$S^1/\{\pm 1\}$ = "不分正反的方向集合"**，几何上等同于半圆。

> 我们需要一个**统一**的构造，让有限点和无穷远点平起平坐。

---

## 2. 关键飞跃：升维一格，让点变成"方向"

**核心 idea**：与其在 2 维平面上拼凑无穷远点，不如**升到 3 维**，让平面上的"点"和"方向"都用同一种东西表达 —— **过原点的直线**。

### 2.1 构造：把 $\mathbb{R}^2$ 嵌入 $\mathbb{R}^3$

把欧氏平面 $\mathbb{R}^2$ 放到 $\mathbb{R}^3$ 中**高度为 1** 的位置：
$$\Pi := \{(x, y, 1) \mid x, y \in \mathbb{R}\} \subset \mathbb{R}^3$$

> [!note] 这是个二维仿射平面
> 它就是我们日常的 $xy$ 平面，只不过被"抬高"到 $z=1$ 处，方便后面的投影构造。

### 2.2 每个有限点 ↔ 一条过原点的直线

对平面上任意一点 $P = (x, y, 1) \in \Pi$，
**经过原点 $O = (0,0,0)$ 和 $P$** 的直线是
$$L_P = \{(tx,\ ty,\ t) \mid t \in \mathbb{R}\}$$

```
        z 轴
         ↑
         |        P=(x,y,1)
         |       ●─────── 平面 Π: z=1
         |      /
         |     /  L_P
         |    /
         |   /
         |  /
       O ●───────── y 轴
        /
       /
      x 轴
```

**重要事实**：直线 $L_P$ 上**除原点外的任意一点** $(tx, ty, t)$（$t \neq 0$）都**唯一对应** $\Pi$ 上的同一个点 $P$。

> 因为 $(tx, ty, t)$ 除以 $t$ 得到 $(x, y, 1)$，刚好是 $\Pi$ 上的 $P$。

### 2.3 但还有"额外"的过原点直线 —— 它们就是无穷远点！

> [!important] 关键发现
> $\mathbb{R}^3$ 中过原点的**水平**直线（在 $z=0$ 平面内）
> 永远不会和 $\Pi$（$z=1$ 平面）相交！

例如 $x$ 轴本身：$L = \{(t, 0, 0) \mid t \in \mathbb{R}\}$，它平行于 $\Pi$，没有交点。

```
        z 轴
         ↑
         |
         |─────────────  Π: z=1
         |
         |
         |     ←─────●─────→  z=0 平面内的直线
         O          (永不与 Π 相交)
         |
```

**这些"水平"过原点直线（整条线都躺在 $z=0$ 平面里），都与 $\Pi$ 平行、永不相交，正好对应平面上的"方向"！**

- $x$ 轴（沿 $(1,0,0)$）：与 $\Pi$ 平行 → 对应 $\Pi$ 上"水平向右"的方向
- 沿 $(1, k, 0)$ 的过原点直线：与 $\Pi$ 平行 → 对应 $\Pi$ 上斜率为 $k$ 的方向

> [!tip] 「平行」与「对应」的区别
> - **平行**：描述 $\mathbb{R}^3$ 里这条过原点直线 与 $\Pi$ 平面 的**几何关系**（永不相交）。
> - **对应**：描述这条直线 与 $\Pi$ 上的**某个无穷远方向** 的**映射关系**（一一对应）。
>
> 正因为它和 $\Pi$ **平行**（不相交），它才不会落到 $\Pi$ 上的某个有限点，从而**对应**一个无穷远点（方向）。

> [!success] 飞跃达成
> $\mathbb{R}^3$ 中**所有过原点的直线** = 平面上**所有点**（有限的 + 无穷远的）
> - $z \neq 0$ 的过原点直线 → 平面上的有限点
> - $z = 0$ 的过原点直线 → 平面上的无穷远点（方向）

## 3. 严格定义：$\mathbb{RP}^2 := (\mathbb{R}^3 \setminus \{0\})\,/\sim$

现在把 §2 的几何观察翻译成严格的代数定义。

### 3.1 等价关系 $\sim$

在 $\mathbb{R}^3 \setminus \{0\}$ 上定义：
$$(x_1, y_1, z_1) \sim (x_2, y_2, z_2)
\quad\Longleftrightarrow\quad
\exists\,\lambda \neq 0 :\ (x_2, y_2, z_2) = \lambda (x_1, y_1, z_1)$$

> [!note] 这正是"同一条过原点直线"
> $(x_1,y_1,z_1)$ 和 $(\lambda x_1, \lambda y_1, \lambda z_1)$ 在同一条过原点直线上 $\iff$ 它们等价。

> [!info] 怎么读懂 (x₁,y₁,z₁) ∼ (x₂,y₂,z₂) 这个关系
>
> 用公式写出来就是：
>
> $$(x_1, y_1, z_1) \sim (x_2, y_2, z_2) \quad\Longleftrightarrow\quad \exists\,\lambda \neq 0 :\ (x_2, y_2, z_2) = \lambda (x_1, y_1, z_1)$$
>
> **一句话**：两个三元组"算同一个东西"，当且仅当它们**差一个非零倍数**。
>
> **为什么要定义这个关系？**
>
> 回到 §2 的几何观察：
> ```
>    一条过原点的直线（除去原点）上有【无穷多个】三元组：
>    (1, 2, 3), (2, 4, 6), (-1, -2, -3), (0.5, 1, 1.5), ...
>           ↓
>    但它们描述的是【同一条直线】= 射影平面上的【同一个点】
> ```
> $\sim$ 就是用来"**告诉计算机：这些三元组应该被视为同一个**"。
>
> **代数 ↔ 几何 对照**
>
> | 代数表达 | 几何含义 |
> | ---- | ---- |
> | $\exists\,\lambda \neq 0 :\ \vec{v_2} = \lambda \vec{v_1}$ | 两个非零向量**共线**（同在一条过原点直线上） |
> | $\lambda > 0$ | 同方向半射线上的点 |
> | $\lambda < 0$ | 反方向半射线上的点（合起来才是完整的过原点直线） |
> | $\lambda \neq 0$（不能为 0） | 排除"任何向量都和 0 等价"的退化情况 |
>
> **它确实是一个合格的"等价关系"**（满足三大铁律）
>
> | 性质 | 验证 |
> | ---- | ---- |
> | **自反性** $a \sim a$ | 取 $\lambda = 1$ ✓ |
> | **对称性** $a \sim b \Rightarrow b \sim a$ | 若 $b = \lambda a$，则 $a = \frac{1}{\lambda} b$（这就是为什么要求 $\lambda \neq 0$）✓ |
> | **传递性** $a \sim b,\ b \sim c \Rightarrow a \sim c$ | 若 $b = \lambda_1 a,\ c = \lambda_2 b$，则 $c = \lambda_1\lambda_2 \cdot a$ ✓ |
>
> **程序员视角：$\sim$ = 给三元组重定义 `__eq__`**
>
> ```python
> # 不再按"三个分量都相同"来比较，而是按"比例是否相同"来比较
> def equivalent(p, q):
>     # 存在非零 λ 使得 q = λ · p
>     # 几何含义：p、q 在同一条过原点直线上
>     ...
>
> equivalent((1, 2, 3), (2, 4, 6))    # True   λ = 2
> equivalent((1, 2, 3), (-1, -2, -3)) # True   λ = -1（反向也算）
> equivalent((1, 2, 3), (1, 2, 4))    # False  比例不同，不共线
> ```
>
> **它正是齐次坐标 $[x:y:z]$ 中"冒号写法"的来源**
> $$[1:2:3] = [2:4:6] = [-1:-2:-3] = [0.5:1:1.5] = \cdots$$
> 方括号 + 冒号是在提醒读者：**只关心比例，不关心具体数值**。

> [!info] 怎么读懂 ℝ³ \ {0} / ∼ 这个记号
>
> 这个看起来"吓人"的符号，其实就是把 §2 的几何观察用代数语言精确表达。可以**逐个符号**翻译：
>
> | 数学符号 | 读法 | 程序员翻译 |
> | ---- | ---- | ---- |
> | $\mathbb{R}^3$ | 三维实数空间 | `Tuple[float, float, float]` 的所有可能取值 |
> | $\setminus \{0\}$ | "扣除"原点 | 排除 `(0, 0, 0)` |
> | $/\sim$ | 按等价关系 $\sim$ "商掉" | 把 $\sim$ 关系下的点合并成同一个 |
>
> **整体含义**：取出 $\mathbb{R}^3$ 中**除原点外**的所有点，再把**互为非零倍数**的点**合并成一个**。
>
> **程序员视角的伪代码**
> ```python
> # 第 1 步：R³ \ {0}
> all_points = {(x, y, z) for x, y, z in R³ if (x, y, z) != (0, 0, 0)}
>
> # 第 2 步：定义等价关系 ~
> def equivalent(p, q):
>     # 存在非零 λ 使得 q = λ·p
>     return exists λ ≠ 0 such that q == (λ*p[0], λ*p[1], λ*p[2])
>
> # 第 3 步：/ ~  即按等价关系分组（求商集）
> RP² = group_by(all_points, equivalent)
> # 每个分组就是一条过原点的直线（去掉原点）
> # RP² 的"一个点" = "一个分组" = "一条过原点直线"
> ```
>
> **三个符号各自的几何意义**
> - $\mathbb{R}^3$：所有 3 元向量构成的空间（升维后的容身之所）
> - $\setminus \{0\}$：原点单独一点没有方向，必须排除（不然 $\lambda \cdot 0 = 0$ 让等价关系崩溃）
> - $/\sim$：把"差一个非零倍数"的点视为同一点（一条过原点直线 ↔ 一个等价类 ↔ 射影平面上的一个点）
>
> 一句话：**$\mathbb{R}^3 \setminus \{0\}/\sim$ = "所有过原点直线"的集合**，就是 §2.3 的几何描述的代数化。
>
>> [!tip] 与 S¹/{±1} 对比记忆
>
> 与一维情形 $S^1/\{\pm 1\}$ 的构造对比，更容易理解 $\mathbb{R}^3 \setminus \{0\}/\sim$：
>
> 两者用的都是同一种数学构造（**商集**：把某些元素视为相同）：
> - $S^1/\{\pm 1\}$：单位圆上**对径点**视为同一 → 得到"方向集合" → **$\mathbb{RP}^1$**（一维射影空间）
> - $\mathbb{R}^3 \setminus \{0\}/\sim$：3 维空间中**同一过原点直线上**的点视为同一 → 得到"过原点直线集合" → **$\mathbb{RP}^2$**（二维射影空间）
>
> 二者都是"把一族对应同一对象的元素压成一个点"的标准操作。

### 3.2 商集即射影平面

$$\boxed{\ \mathbb{RP}^2 := (\mathbb{R}^3 \setminus \{0\}) / \sim\ }$$

> [!info] 三个等价的几何描述
> $\mathbb{RP}^2$ 的"点"可以被理解为：
> 1. $\mathbb{R}^3 \setminus \{0\}$ 中按 $\sim$ 划分的**等价类**
> 2. $\mathbb{R}^3$ 中所有**过原点的直线**
> 3. $S^2$ 上**对径点对**（每对反向单位向量）

### 3.3 "齐次坐标"$[x : y : z]$

每个等价类用方括号记号表示：
$$[x : y : z] := \{(\lambda x, \lambda y, \lambda z) \mid \lambda \neq 0\}$$

> [!tip] 重要约定
> - 冒号表示**比值**而非数值
> - $[x:y:z] = [\lambda x : \lambda y : \lambda z]$ 对任意 $\lambda \neq 0$ 成立
> - $[0:0:0]$ **未定义**（因为已从 $\mathbb{R}^3$ 中扣除原点）

### 3.4 有限点 vs. 无穷远点

| 类型 | 齐次坐标 | 对应几何对象 |
| ---- | ---- | ---- |
| 有限点 | $[x:y:1]$（或任意 $z\neq 0$ 的代表） | 平面上点 $(x/z, y/z)$ |
| 无穷远点 | $[x:y:0]$ | 方向 $(x,y)$（与平行线族对应） |

> [!example] 程序员视角的转换公式
> ```python
> # 齐次 → 笛卡尔
> def homogeneous_to_cartesian(x, y, z):
>     if z == 0:
>         return f"infinity in direction ({x}, {y})"
>     return (x / z, y / z)
> 
> # 笛卡尔 → 齐次（最简单的选择）
> def cartesian_to_homogeneous(x, y):
>     return [x, y, 1]
> ```

### 3.5 小结：怎么完整地"读懂"射影平面？

> [!summary] 射影平面 ℝP² 的 5 个等价视角
> 
> （即 $\mathbb{RP}^2$，下同。）同一个对象、5 种表述，**任选一种都对，遇到具体问题时挑最顺手的那种用**：
> 
> | 视角 | $\mathbb{RP}^2$ 的"点" | 适合解决的问题 |
> |:-:|:-|:-|
> | ① **代数定义** | 等价类 $[x:y:z]$，其中 $(x,y,z)\sim(\lambda x,\lambda y,\lambda z)$ | 工程实现、写代码、矩阵运算 |
> | ② **几何模型 A** | $\mathbb{R}^3$ 中**过原点的直线** | 想"它长什么样"时的最直观图像 |
> | ③ **几何模型 B** | 单位球面 $S^2$ 上**对径点对** $\{P, -P\}$ | 想要紧致流形时的标准模型 |
> | ④ **拼贴模型** | 笛卡尔平面 $\mathbb{R}^2$ **+** 一圈无穷远点（每个方向一个） | 直观理解"为什么平行线相交" |
> | ⑤ **方向集合** | $\mathbb{R}^3$ 中所有"过原点方向"的集合 | 在相机/投影/光线追踪里的语义 |

#### 因果链条一图穿起来

```
   想要的性质：任意两条不同直线交于唯一一点（无例外）
                  ↓
   笛卡尔平面缺了什么：平行线没有交点
                  ↓
   补什么：每个方向补一个"无穷远点"
                  ↓
   怎么补得自然：把 ℝ² 嵌入 ℝ³ 的 z=1 平面，每个点拉成过原点直线
                  ↓
   形式化：ℝP² := (ℝ³ ∖ {0}) / ∼     ← §3 核心定义
                  ↓
   坐标系统：齐次坐标 [x:y:z]         ← §3.3
                  ↓
   工程红利：相交、变换、投影统一为线性代数（叉积、矩阵）  ← §5
```

#### 一句话总结（写在便签上的版本）

> $\mathbb{RP}^2$ = **欧氏平面 + 一圈无穷远点**，本质上是 $\mathbb{R}^3$ 中**所有过原点直线**的集合。  
> 它把"平行 / 不平行"的二分消除，把"相交"变成无例外的全局定理。

### 3.6 扩展：$\mathbb{RP}^n$ 是一族空间，不止 $\mathbb{RP}^2$

§3 的所有构造**与维度无关**，把 $\mathbb{R}^3$ 换成 $\mathbb{R}^{n+1}$，整套定义照搬不误。

#### 通用定义

$$\boxed{\;\mathbb{RP}^n \;:=\; (\mathbb{R}^{n+1} \setminus \{0\})\,/\sim,\quad \text{其中 } \mathbf x \sim \lambda \mathbf x,\;\lambda \neq 0\;}$$

> [!note] 维度的"+1"在哪里？
> $\mathbb{RP}^n$ 是 **n 维**射影空间，但用的坐标是 **(n+1) 维齐次坐标** $[x_0:x_1:\cdots:x_n]$。
> "+1" 就是为了表达"无穷远"那一维 —— 和 $\mathbb{RP}^2$ 用 3 个数表示 2 维平面上的点完全同构。

#### 常见的几个

| 空间 | 维度 | 齐次坐标 | 几何对象 | 典型应用 |
|:-:|:-:|:-:|:-|:-|
| $\mathbb{RP}^1$ | 1 维 | $[x:y]$ | 实数轴 + 1 个无穷远点（拓扑上是**圆** $S^1$） | 透视下的 1D 投影、CRT 中的色品轴 |
| $\mathbb{RP}^2$ | 2 维 | $[x:y:z]$ | 平面 + 一圈无穷远点（每方向一个） | 计算机视觉、2D 仿射/射影变换 |
| $\mathbb{RP}^3$ | 3 维 | $[x:y:z:w]$ | 空间 + 一张"无穷远平面" | **3D 图形学**（OpenGL/DirectX 的 4×4 矩阵）、机器人位姿 |
| $\mathbb{RP}^n$ | $n$ 维 | $[x_0:\cdots:x_n]$ | $\mathbb{R}^n$ + 一个 $\mathbb{RP}^{n-1}$ 的"无穷远超平面" | 高维几何、代数几何 |

> [!example] $\mathbb{RP}^3$ 在你电脑里随时在跑
> 打开任意 3D 游戏 / CAD / 渲染器，每个顶点都被表示为 4 维齐次坐标 $(x,y,z,w)$：
> 
> - **平移**（一般 3D 仿射变换无法用 3×3 矩阵表达）→ 升维到 $\mathbb{RP}^3$ 后，就能用一个 **4×4 矩阵** 统一处理"旋转 + 缩放 + 平移 + 透视"；
> - **透视除法**（光栅化阶段的 `xyz / w`）→ 就是把 $[x:y:z:w]$ 折回笛卡尔 $(x/w, y/w, z/w)$ —— 完全是 §3.4 转换公式的 3D 版本。
> 
> 换句话说：你写 `glm::mat4` 或 `DirectX::XMMATRIX` 时，**已经在用 $\mathbb{RP}^3$ 了**，只是没人告诉你这个名字。

#### 还有别的扩展吗？

> [!info] 除了改维度 $n$，还能改"标量域"
> 
> 把基底从实数 $\mathbb{R}$ 换成别的域，会得到一族"亲戚"：
> 
> | 名字 | 定义 | 备注 |
> |:-:|:-|:-|
> | $\mathbb{CP}^n$ | $(\mathbb{C}^{n+1} \setminus \{0\})/\sim$ | **复射影空间**。$\mathbb{CP}^1$ 拓扑上是球面 $S^2$（黎曼球面），是复分析、量子力学（Bloch 球）的核心 |
> | $\mathbb{HP}^n$ | 用四元数 $\mathbb H$ 替换 | 四元数射影空间，出现在李群、规范场论中 |
> | $\mathrm{Gr}(k, n)$ | $\mathbb{R}^n$ 中所有 $k$ 维子空间的集合 | **Grassmann 流形**，$\mathrm{Gr}(1, n+1) = \mathbb{RP}^n$ 是其特例（"过原点直线" = 1 维子空间） |
> | $\mathbb{F}_q\mathbb P^n$ | 用有限域 $\mathbb{F}_q$ 替换 | 有限射影几何，编码理论、组合设计的工具 |
> 
> 程序员日常 99% 的需求都在 $\mathbb{RP}^2$（图像）和 $\mathbb{RP}^3$（3D 图形）之内；其余几个是数学家和理论物理学家的工具。

## 4. 推导：为什么平行线在 $\mathbb{RP}^2$ 中相交？

这是齐次坐标最漂亮的应用。我们一步一步推导。

### 4.1 直线在齐次坐标下的方程

> [!info] 先复习：什么是"齐次坐标"？
>
> **一句话定义**：用 **3 个数** $(X, Y, Z)$ 表示平面上**一个点**，并且**整体放缩任意非零倍**仍代表同一点。
>
> $$[X : Y : Z] \;=\; [\lambda X : \lambda Y : \lambda Z],\quad \lambda \neq 0$$
>
> **与笛卡尔坐标 $(x, y)$ 的换算**（详见 §3.4）：
>
> | 类型 | 笛卡尔 $(x,y)$ | 齐次 $[X:Y:Z]$ | 换算公式 |
> |------|---------------|----------------|---------|
> | **普通点** | $(x, y)$ | $[x : y : 1]$ | $x = X/Z,\ y = Y/Z$（$Z \neq 0$） |
> | **无穷远点** | 不存在 | $[X : Y : 0]$ | $Z = 0$ → 表示"方向 $(X, Y)$" |
>
> **冒号记号 `[X : Y : Z]`** = 强调"只关心比例，不关心绝对大小"，例如 $[2:4:1] = [4:8:2] = [1:2:0.5]$ 是同一个点。
>
> **为什么要多加一个 $Z$？**
> - $Z \neq 0$：表示"有限远的普通点"
> - $Z = 0$：表示"无穷远的方向点"
> - → 一套坐标统一处理普通点和无穷远点，这就是齐次坐标的威力。

平面上一条直线 $ax + by + c = 0$（$a, b$ 不同时为零）。

把 $(x, y) = (X/Z,\ Y/Z)$（$Z \neq 0$）代入：
$$a \cdot \frac{X}{Z} + b \cdot \frac{Y}{Z} + c = 0$$

两边乘 $Z$：
$$\boxed{\ aX + bY + cZ = 0\ }$$

> [!info] 什么是"齐次方程"？
>
> **一句话定义**：方程中**所有项的次数都相同**，称为"齐次方程"（Homogeneous Equation）。
>
> **判别准则**（最实用）：把变量整体放缩 $\lambda$ 倍，方程是否仍然成立？
>
> $$F(X, Y, Z) = 0 \quad \stackrel{?}{\Longleftrightarrow} \quad F(\lambda X, \lambda Y, \lambda Z) = 0,\ \forall \lambda \neq 0$$
>
> **对照表**：
>
> | 方程 | 各项次数 | 是否齐次 | 放缩验证 |
> |------|---------|---------|---------|
> | $aX + bY + cZ = 0$ | 都是 1 次 | ✅ 齐次 | $a(\lambda X)+b(\lambda Y)+c(\lambda Z) = \lambda(aX+bY+cZ) = 0$ ✓ |
> | $X^2 + Y^2 - Z^2 = 0$ | 都是 2 次 | ✅ 齐次 | 整体出现因子 $\lambda^2$，原方程为 0 仍为 0 ✓ |
> | $ax + by + c = 0$ | 1 次、1 次、**0 次** | ❌ 不齐次 | 常数项 $c$ 不随 $\lambda$ 缩放 ✗ |
> | $X^2 + Y - Z = 0$ | 2 次、1 次、1 次 | ❌ 不齐次 | 各项缩放因子不同 ✗ |
>
> **为什么射影几何只接受齐次方程？**
>
> 射影点 $[X:Y:Z]$ 本身不是单点，而是**一整条过原点直线**上的所有点 $\{(\lambda X, \lambda Y, \lambda Z) \mid \lambda \neq 0\}$。
>
> 要让"方程在该点上成立"这件事**不依赖于选了直线上哪个代表元**，方程必须满足：
>
> $$F(X, Y, Z) = 0 \;\Longrightarrow\; F(\lambda X, \lambda Y, \lambda Z) = 0$$
>
> ↑ 这正是**齐次方程**的定义。
>
> → **结论**：齐次坐标 + 齐次方程 = 与等价关系 $\sim$ 完美兼容的代数系统。

> [!success] 漂亮的事实：笛卡尔 → 齐次的"升次"
>
> 笛卡尔方程 $ax+by+c=0$（**非齐次**，常数项 $c$ 是 0 次）
>
> 在齐次坐标下变为 $aX+bY+cZ=0$（**齐次**，所有项都是 1 次）
>
> —— 引入 $Z$ 把"常数项"补成了 1 次项 $cZ$，整个方程"升次"为齐次！这正是齐次坐标的精妙之处。

### 4.2 直线在 $\mathbb{RP}^2$ 中是什么？

齐次方程 $aX+bY+cZ=0$ 在 $\mathbb{R}^3$ 里描述的是**过原点的平面**。

```
       z 轴
        ↑
        |  /─────────── 平面 Π: z=1
        | /            ↑这里是普通直线
        |/   斜切的
        ●──── 过原点平面 (aX+bY+cZ=0)
       /|
      / |
        |
```

- 过原点平面 $\cap$ $\Pi$ = 普通直线（$z=1$ 平面上的截线）
- 过原点平面 $\cap$ $\{z=0\}$ = 一条过原点直线 = **该方向上的无穷远点**

> [!note] 一条射影直线 = 过原点的平面
> 射影直线包含：
> - 一条普通直线（无穷多个有限点）
> - 加上**一个**无穷远点（对应这条直线的方向）

### 4.3 推导：两条平行线相交于哪里？

设两条平行直线（笛卡尔形式）：
$$L_1:\ ax + by + c_1 = 0$$
$$L_2:\ ax + by + c_2 = 0,\quad c_1 \neq c_2$$

> 它们斜率相同（前两项系数相同），但截距不同 → 平行不相交。

**齐次化**：
$$L_1:\ aX + bY + c_1 Z = 0$$
$$L_2:\ aX + bY + c_2 Z = 0$$

**求交点**：联立两个方程，相减消去前两项：
$$(c_1 - c_2)\, Z = 0$$

由于 $c_1 \neq c_2$，必有
$$\boxed{\ Z = 0\ }$$

代回任一方程：
$$aX + bY = 0\ \Longrightarrow\ (X, Y) \propto (b, -a)$$

> [!success] 推导结果
> 两条平行线的**唯一**交点是
> $$[X : Y : Z] = [b : -a : 0]$$
> 这是个**无穷远点**（$Z=0$），方向恰好是 $(b, -a)$ —— 即直线 $ax+by+c=0$ 的方向向量！

### 4.4 几何直观：为什么交点恰好在 $Z=0$？

回到 §2 的"升维"图像：

- 直线 $L_1$ 在 $\Pi$（$z=1$ 平面）上 → 它对应的过原点平面 $\Pi_1$ 切过 $L_1$
- 直线 $L_2$ 在 $\Pi$ 上 → 对应过原点平面 $\Pi_2$
- $L_1 \parallel L_2$ → $\Pi_1$ 和 $\Pi_2$ 在 $\Pi$ 上**平行截痕**

> [!info] 怎么理解这里的"升维"图像？
> 
> **核心动作**：把笛卡尔平面 $\Pi$（$z=1$）上的**一条直线** $L_1$，升级成 $\mathbb{R}^3$ 中的**一张过原点平面** $\Pi_1$。
> 
> | 升维前（笛卡尔世界） | 升维后（射影世界） |
> |:-:|:-:|
> | $L_1$：$\Pi$ 上的一条直线（1 维对象） | $\Pi_1$：$\mathbb{R}^3$ 中的一张过原点平面（2 维对象） |
> | $L_1$ 上的点 $(x, y)$ | 过原点和 $(x, y, 1)$ 的直线 $[x:y:1]$ |
> 
> **"$\Pi_1$ 切过 $L_1$" 是什么意思？**
> 
> = "$\Pi_1$ 包含整条 $L_1$"。
> 想象拿一把刀，沿着 $L_1$ 和原点 $O$ 一起切下去 —— 切出来的那张倾斜平面就是 $\Pi_1$：
> 
> ```
>          z 轴
>           ↑       Π₁（倾斜平面，把 L₁ 嵌进来）
>           │      ╱
>           │     ╱─────●━━━●━━━●─────   ← L₁ 躺在 Π: z=1 里
>           │    ╱      ↑   ↑   ↑
>           │   ╱       │   │   │  ← L₁ 上每点对应一条过 O 的射线
>           │  ╱        │   │   │     这些射线全都躺在 Π₁ 这张平面里
>           │ ╱         │   │   │
>           │╱          │   │   │
>           ●━━━━━━━━━━━━━━━━━━━━━━     ← Π₁ 与 z=0 的交线
>          O                              （这正是 L₁ 的"方向向量"方向）
> ```
> 
> **为什么 $\Pi_1$ 是唯一的？** 因为 $L_1$ 不过原点（它在 $z=1$），所以 $L_1$ 和 $O$ **不共线** → 三维几何里 "一条直线 + 一个不在其上的点" 唯一确定一张平面。
> 
>> [!warning] 易混淆点：升维前后有**两个不同的原点**
>>
>> 上面说"$L_1$ 不过原点"，这里的"原点"指的是 **$\mathbb{R}^3$ 的原点 $O = (0,0,0)$**，**不是**笛卡尔平面的原点 $(0,0)$。这两个原点分属两个坐标系，**不是同一个点**。
>>
>> **两个坐标系对照**
>>
>> | | 升维前（笛卡尔世界） | 升维后（射影世界 / $\mathbb{R}^3$） |
>> |:-:|:-:|:-:|
>> | **空间** | 2D 平面 $\Pi$ | 3D 空间 $\mathbb{R}^3$ |
>> | **坐标** | $(x, y)$ | $(X, Y, Z)$ |
>> | **原点** | $(0, 0)$ | $O = (0, 0, 0)$ |
>> | **维度** | 2 维 | 3 维 |
>>
>> **两个原点的几何关系**
>>
>> 笛卡尔平面 $\Pi$ 被整体抬高到 $z=1$ 高度，所以：
>>
>> ```
>>                z 轴
>>                 ↑
>>                 │
>>                 │       Π: z = 1 平面（笛卡尔世界住在这里）
>>        ─────────┼─────●─────────  ← 笛卡尔原点 (0,0)
>>                 │     ↑           升维后位于 ℝ³ 中的 (0, 0, 1)
>>                 │     │
>>                 │     │ ← 高度差 1
>>                 │     │
>>                 │     ↓
>>                 ●─────┼──────── x/y 平面 (z=0)
>>              O=(0,0,0)
>>              ↑
>>              这才是射影世界（ℝ³）的原点
>> ```
>>
>> | 称呼 | 在 $\mathbb{R}^3$ 中的位置 | 角色 |
>> |:-:|:-:|:-:|
>> | 笛卡尔原点 $(0, 0)$ | $(0, 0, 1)$ | $\Pi$ 这张"纸"的中心，是 $\Pi$ 上一个**普通点** |
>> | 射影原点 $O$ | $(0, 0, 0)$ | $\mathbb{R}^3$ 这间"屋子"的中心，**不在 $\Pi$ 上** |
>>
>> 二者**沿 z 轴相差 1 个单位**，绝不是同一个点。
>>
>> **为什么必须区分？—— §4.4 的论证依赖于此**
>>
>> 1. $L_1$ 整体躺在 $z=1$ 平面里 → $L_1$ 上每个点的 $z$ 坐标都是 $1$，**不可能是 $0$** → $L_1$ 一定**不经过射影原点 $O$**；
>> 2. 因此 "$L_1$ + $O$" = "一条直线 + 一个不在其上的点" → 唯一确定一张平面 $\Pi_1$；
>> 3. 这一步**与 $L_1$ 是否经过笛卡尔原点 $(0,0)$ 无关**：即使 $L_1$ 过笛卡尔原点（如 $y=2x$），它升维后也只是经过 $(0,0,1)$，依然不过 $O$，论证仍然成立。
>>
>> **用齐次坐标再确认一次**
>>
>> | 笛卡尔点 | 升维到 $\mathbb{R}^3$ 中的位置 | 齐次坐标 |
>> |:-:|:-:|:-:|
>> | 笛卡尔原点 $(0, 0)$ | $(0, 0, 1)$ | $[0 : 0 : 1]$ ✓ 合法射影点 |
>> | 射影原点 $O$ | $(0, 0, 0)$ | $[0:0:0]$ ✗ **非法**（齐次坐标禁止全零） |
>>
>> 这正是 §3 中 $\mathbb{RP}^2 = (\mathbb{R}^3 \setminus \{0\})/\sim$ 必须把 $\{0\}$ 挖掉的根本原因 —— 射影原点 $O$ 在射影世界里**不代表任何一点**，它只是所有过原点直线的公共汇聚处，是升维容器的"几何枢纽"。
>>
>> 一句话记忆：**笛卡尔原点是 $\Pi$ 这张"纸"的中心；射影原点 $O$ 是 $\mathbb{R}^3$ 这间"屋子"的中心。纸悬浮在屋子中央上方 1 米处。**
> 
> **为什么必须是"过原点平面"？** 回忆 §3 的核心定义：
> 
> - 射影平面里的**一个点** ↔ $\mathbb{R}^3$ 中**一条过原点直线**
> - 射影平面里的**一条直线** ↔ $\mathbb{R}^3$ 中**一张过原点平面**
> 
> $L_1$ 上每个点 $(x, y, 1)$ 都被升维成"过 $O$ 和 $(x, y, 1)$ 的射线"，**所有这些射线集体居住的那张平面**，正是 $\Pi_1$。
> 
> **这个升维妙在哪？**  
> 笛卡尔世界里，$L_1 \parallel L_2$ → **永不相交**（二维平面的窘境）。  
> 升维后，$\Pi_1$ 和 $\Pi_2$ 是**两张不同的过原点平面** → 三维几何定理保证 **必然相交于一条过原点直线**。  
> "找不到交点" 被翻译成 "必然有交线"，无穷远点就这样被自然挤了出来 —— 下一段会看到，这条交线恰好躺在 $z=0$ 上。

> [!tip] 关键几何事实
> 两个**不同**的过原点平面**必然相交于一条过原点直线**
> （三维空间中两个相交平面的交线定理）
> 
> 当 $L_1 \parallel L_2$ 时，这条交线**不可能**穿过 $\Pi$（否则就是 $L_1, L_2$ 的有限交点，矛盾），
> 所以**这条交线必然躺在 $z=0$ 平面里** —— 这就是无穷远点！

```
          Π_1 (倾斜平面)
           ╲       
            ╲   Π_2 (另一倾斜平面)
        ─────╲────────  Π: z=1
         L_1  ╲   L_2     ← 平面上看是两条平行线
              ╲╲
               ╲╲←── 两个平面交于过原点的水平直线
        ━━━━━━━━╲━━━━━━━━ z=0
                 ●── 这条交线 = 无穷远点
                 O
```

### 4.5 不平行的情况：交点自动落在有限处

设两条**不平行**直线：
$$L_1:\ a_1 X + b_1 Y + c_1 Z = 0$$
$$L_2:\ a_2 X + b_2 Y + c_2 Z = 0$$

不平行 $\iff$ $a_1 b_2 - a_2 b_1 \neq 0$。

用 Cramer 法则解三元齐次方程组（取 $Z$ 为参数），最终得到：
$$[X : Y : Z] = [\,b_1 c_2 - b_2 c_1\ :\ c_1 a_2 - c_2 a_1\ :\ a_1 b_2 - a_2 b_1\,]$$

由于 $a_1 b_2 - a_2 b_1 \neq 0$，所以 $Z \neq 0$ —— **交点自动是有限点**。

> [!success] 统一定理
> $\mathbb{RP}^2$ 中**任意两条不同的射影直线恰好交于一点**：
> - 不平行 → 交点 $Z \neq 0$（有限点）
> - 平行   → 交点 $Z = 0$（无穷远点）
> 
> "平行线相交"不再是诡辩，而是定理。

## 5. 把外积一并讲清楚（程序员最爱）

> [!abstract] 本节核心结论（先看后推）
> 在齐次坐标下，**两点连线** 和 **两线交点** 都归结为同一个三维向量运算 —— **叉积（外积）**：
> 
> $$\boxed{\;\ell = P_1 \times P_2 \quad,\quad P = \ell_1 \times \ell_2\;}$$
> 
> 一个 `np.cross` 调用，**自动处理**所有特殊情况（平行、重合、过原点、无穷远……）。这正是齐次坐标值得引入的实践理由。

---

### 5.1 准备：点-线关联方程

回忆 §4.1：在齐次坐标下，"点 $P$ 在直线 $\ell$ 上" 等价于一个对称的内积方程：

$$\ell \cdot P \;=\; aX + bY + cZ \;=\; 0
\qquad
\text{其中 } P=\begin{pmatrix}X\\Y\\Z\end{pmatrix},\;
\ell=\begin{pmatrix}a\\b\\c\end{pmatrix}$$

这条关联方程是后面所有推导的种子 —— 它对 $P$ 和 $\ell$ **完全对称**，这正是"对偶性"的根源。

---

### 5.2 推导一：两点定一线 → 为什么是外积？

**问题**：已知两点 $P_1, P_2 \in \mathbb{RP}^2$，求过它们的直线 $\ell$。

**翻译成方程**：要找 $\ell=(a,b,c)^\top$ 同时满足
$$\ell \cdot P_1 = 0 \quad \text{且} \quad \ell \cdot P_2 = 0$$

**几何解读**（关键一步）：

| 代数语言 | 三维几何语言 |
|:-|:-|
| $\ell \cdot P_1 = 0$ | 向量 $\ell$ **垂直于** $P_1$ |
| $\ell \cdot P_2 = 0$ | 向量 $\ell$ **垂直于** $P_2$ |
| 同时垂直于 $P_1, P_2$ | $\ell$ 沿 $P_1, P_2$ **法向量**方向 |

而"求两个向量的公共法向量"，正是**叉积**的几何定义：

$$\boxed{\;\ell \;=\; P_1 \times P_2\;}$$

> [!note] 为什么不需要解方程？
> 表面上要解一个 $2\times 3$ 线性方程组（两条方程、三个未知数），齐次解空间是 1 维的（恰好对应"齐次坐标只在比例意义下唯一"）。叉积公式 $P_1\times P_2$ **直接给出这条 1 维解空间的一个代表向量**，省去了高斯消元。

---

### 5.3 推导二：两线定一点 → 完全对称

**问题**：已知两线 $\ell_1, \ell_2$，求交点 $P$。

**翻译成方程**：要找 $P=(X,Y,Z)^\top$ 同时满足
$$\ell_1 \cdot P = 0 \quad \text{且} \quad \ell_2 \cdot P = 0$$

把上一节的 $\ell\leftrightarrow P$ 角色一换，结论立刻镜像得到：

$$\boxed{\;P \;=\; \ell_1 \times \ell_2\;}$$

> [!tip] 射影对偶性（Duality）—— 一行话理解
> 把所有"点"换成"线"、"线"换成"点"，所有定理依然成立。
> - "两**点**定一**线**"  ↔  "两**线**定一**点**"
> - 公式同为 $\times$（外积），不需要任何修改。
> 
> 这是 $\mathbb{RP}^2$ 比欧氏平面更"对称漂亮"的地方 —— 欧氏世界里，平行线没有交点，对偶性就被破坏了。

### 5.4 例子 A：两点定一线（对应 §5.2 推导）

> [!example] 求过两点 P1=(1,2) 与 P2=(3,4) 的直线
> 
> **Step 1：升维到齐次坐标**
> 
> 笛卡尔点 $(x,y)$ → 齐次点 $[x:y:1]$（取最自然的代表 $Z=1$）：
> 
> | 笛卡尔点 | 齐次坐标 $P$ |
> |:-:|:-:|
> | $(1, 2)$ | $P_1 = (1,\; 2,\; 1)$ |
> | $(3, 4)$ | $P_2 = (3,\; 4,\; 1)$ |
> 
> **Step 2：算外积** $\ell = P_1 \times P_2$
> $$\ell = \begin{vmatrix}\mathbf i & \mathbf j & \mathbf k \\ 1 & 2 & 1 \\ 3 & 4 & 1\end{vmatrix}
> = \big(2\cdot 1 - 1\cdot 4,\; 1\cdot 3 - 1\cdot 1,\; 1\cdot 4 - 2\cdot 3\big)
> = (-2,\; 2,\; -2)$$
> 
> 化简（齐次坐标只在比例意义下唯一，整体除以 $-2$）：
> $$\ell = (-2,\;2,\;-2) \;\sim\; (1,\;-1,\;1)$$
> 
> **Step 3：翻译回笛卡尔方程**
> 
> 齐次直线 $\ell=(a,b,c)=(1,-1,1)$ 对应方程 $aX+bY+cZ=0$，再令 $Z=1$：
> $$X - Y + 1 = 0 \;\;\Longrightarrow\;\; y = x + 1$$
> 
> 验算：$(1,2)$ → $2=1+1$ ✓；$(3,4)$ → $4=3+1$ ✓。
> 
> ```python
> import numpy as np
> P1 = np.array([1, 2, 1])
> P2 = np.array([3, 4, 1])
> L = np.cross(P1, P2)             # [-2, 2, -2]
> a, b, c = L
> print(f"直线方程: {a}x + {b}y + {c} = 0")   # -2x + 2y - 2 = 0  即 y = x + 1
> ```

> [!note] 几何理解
> $\ell = P_1\times P_2$ 给出的是 $\mathbb R^3$ 中由 $P_1,P_2$ 两个三维向量所张成平面的**法向量**。  
> 而这张平面与 $Z=1$ 平面的**交线**，正是笛卡尔世界中那条经过 $(1,2),(3,4)$ 的直线。  
> ——这就是 §4.4「升维-切片」图像在两点定线中的具体演练。

### 5.5 例子 B：一般情况（两条普通直线相交）

> [!example] 求直线 y = x 与 y = -x + 2 的交点
> 
> **Step 1：化为齐次形式**
> 
> | 笛卡尔方程 | 齐次方程 | 齐次系数 $\ell$ |
> |:-:|:-:|:-:|
> | $y = x$ | $X - Y = 0$ | $\ell_1 = (1, -1, 0)$ |
> | $y = -x + 2$ | $X + Y - 2Z = 0$ | $\ell_2 = (1, 1, -2)$ |
> 
> **Step 2：算外积**
> $$\ell_1 \times \ell_2 = \begin{vmatrix}\mathbf i & \mathbf j & \mathbf k \\ 1 & -1 & 0 \\ 1 & 1 & -2\end{vmatrix}
> = \big((-1)(-2)-0\cdot 1,\; 0\cdot 1-1\cdot(-2),\; 1\cdot 1-(-1)\cdot 1\big)
> = (2,\; 2,\; 2)$$
> 
> **Step 3：反归一化回笛卡尔**
> 
> $Z=2 \neq 0$，是**有限点**。除以 $Z$：
> $$P = [2:2:2] = [1:1:1] \;\;\Longrightarrow\;\; (x,y) = (1, 1)\;\checkmark$$
> 
> 验算：$y=x$ 给 $1=1$ ✓；$y=-x+2$ 给 $1 = -1+2$ ✓。
> 
> ```python
> import numpy as np
> L1 = np.array([1, -1,  0])
> L2 = np.array([1,  1, -2])
> P = np.cross(L1, L2)             # [2, 2, 2]
> print(P[:2] / P[2])              # [1. 1.]   ← 笛卡尔交点
> ```

### 5.6 例子 C：平行线 → 交于无穷远点

> [!example] 求平行线 x+y=1 与 x+y=2 的"交点"
> 
> **Step 1：化为齐次形式**
> 
> | 笛卡尔方程 | 齐次方程 | 齐次系数 $\ell$ |
> |:-:|:-:|:-:|
> | $x+y=1$ | $X+Y-Z=0$ | $\ell_1 = (1, 1, -1)$ |
> | $x+y=2$ | $X+Y-2Z=0$ | $\ell_2 = (1, 1, -2)$ |
> 
> **Step 2：算外积**
> $$\ell_1 \times \ell_2 = \begin{vmatrix}\mathbf i & \mathbf j & \mathbf k \\ 1 & 1 & -1 \\ 1 & 1 & -2\end{vmatrix}
> = \big(1\cdot(-2)-(-1)\cdot 1,\; (-1)\cdot 1-1\cdot(-2),\; 1\cdot 1-1\cdot 1\big)
> = (-1,\; 1,\; 0)$$
> 
> **Step 3：解读结果**
> 
> $Z=0$ —— **无穷远点**！代表方向 $(-1, 1)$，即 $y=-x$ 的方向。
> 
> 这恰好是原平行线 $x+y=\text{常数}$ 的方向（方向向量满足 $1\cdot dx + 1\cdot dy = 0$，即 $dy = -dx$）。✓
> 
> ```python
> import numpy as np
> L1 = np.array([1, 1, -1])
> L2 = np.array([1, 1, -2])
> P = np.cross(L1, L2)             # [-1, 1, 0]
> if P[2] == 0:
>     print(f"平行！方向 = ({P[0]}, {P[1]})")   # 方向 = (-1, 1)
> else:
>     print(P[:2] / P[2])
> ```
> 
> **算法的优雅**：写代码时**完全不用 `if 平行 then ... else ...`**。同一行 `np.cross` 既处理"两线相交于有限点"，也处理"两线平行于无穷远点"，全凭 $Z$ 是否为 0 自动区分。

### 5.7 一张表收尾

| 任务 | 公式 | 几何解释 | 退化情形 | 对应例子 |
|:-|:-:|:-|:-|:-:|
| 两点 → 直线 | $\ell = P_1 \times P_2$ | $\ell$ 是 $P_1, P_2$ 在 $\mathbb R^3$ 中张成平面的法向量 | $P_1 \parallel P_2$（同点）→ $\ell=0$，无定义 | A |
| 两线 → 交点 | $P = \ell_1 \times \ell_2$ | $P$ 是两个法平面的交线方向 | $\ell_1 \parallel \ell_2$（同线）→ $P=0$，无定义 | B / C |
| 点-线关联 | $\ell \cdot P = 0$ | $\ell \perp P$（在 $\mathbb R^3$ 内积意义下） | —— | —— |

> [!summary] 第5章一句话
> 把点和线都升维成 $\mathbb R^3$ 中的向量后，"几何关联"变成"内积为零"，"几何求解"变成"叉积"。一切平面几何的相交问题，都被翻译成了三维向量代数 —— 这就是齐次坐标在工程上无可替代的价值。

## 6. 因果关系总结：为什么必须这样定义？

```
┌─────────────────────────────────────────────────────────┐
│ ① 想让"任意两条不同直线交于一点"无例外成立              │
│             ↓ (需要给平行线添加交点)                     │
│ ② 必须给平面"补"上一些"无穷远点"                        │
│             ↓ (无穷远点的本质是"方向")                   │
│ ③ 方向 = 一族平行线的共同特征                            │
│             ↓ (希望有限点和无穷远点统一表达)             │
│ ④ 升维：把每个点想象成"过原点的直线"                    │
│             ↓ (有限点用 z≠0 直线，无穷远点用 z=0 直线)   │
│ ⑤ "过原点直线" = 比例同类 = 等价类                      │
│             ↓ (代数化)                                   │
│ ⑥ 定义 RP² := (R³ \ {0}) / ~                            │
│             ↓ (验证)                                     │
│ ⑦ 直线方程齐次化 → 求交点 → 平行 ⟺ Z=0                  │
│             ↓                                            │
│ ⑧ 平行线相交于无穷远点成为定理 ✓                        │
└─────────────────────────────────────────────────────────┘
```

> [!quote] 数学家 Felix Klein 的话
> > "射影几何就是几何中的几何。"
>
> $\mathbb{RP}^2 = (\mathbb{R}^3 \setminus \{0\})/\sim$ 这个看似抽象的定义，
> 把"无穷远"从一个**哲学概念**，转化为**坐标可计算**的对象，
> 把欧式几何里的"例外情况"，统一进**线性代数的框架**。
> 
> 这就是齐次坐标"漂亮"的真正含义。

## 7. 给程序员的 take-away

> [!summary] 一页纸总结
> 
> **定义**
> $$\mathbb{RP}^2 = \{[x:y:z] \mid (x,y,z) \in \mathbb{R}^3, \text{不全为零}\},\quad [x:y:z]=[\lambda x:\lambda y:\lambda z]$$
> 
> **几何字典**
> 
> | 射影概念 | 代数表示 | 几何对象 |
> | ---- | ---- | ---- |
> | 点 | $[X:Y:Z]$ | 过原点直线 |
> | 直线 | $aX+bY+cZ=0$ | 过原点平面 |
> | 有限点 | $Z \neq 0$ | 与 $z=1$ 平面相交 |
> | 无穷远点 | $Z = 0$ | 平行于 $z=1$ 平面（一个方向） |
> | 无穷远直线 | $Z = 0$ | $xy$ 平面本身 |
> 
> **核心公式**
> 
> $$\text{两点连线} = P_1 \times P_2$$
> $$\text{两线交点} = \ell_1 \times \ell_2$$
> 
> **核心结论**
> 
> 任意两条不同的射影直线**恰好相交于一点**；
> 平行线交于 $Z=0$ 的无穷远点，方向就是 $(b, -a)$。
