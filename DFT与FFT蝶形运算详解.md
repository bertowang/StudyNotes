# DFT 与 FFT：蝶形运算、码位倒序与频谱分析

---

## 一、DFT 基本概念

### 1.1 定义

离散傅里叶变换（DFT）将长度为 $N$ 的时域序列转换为频域表示：

$$
X[k] = \sum_{n=0}^{N-1} x[n] \cdot W_N^{kn}, \quad k = 0, 1, \dots, N-1
$$

其中**旋转因子** $W_N = e^{-j2\pi/N}$。

### 1.2 频率分量分布

对于实信号，DFT 具有**共轭对称性** $X[k] = X^*[N - k]$：

| $k$ 范围 | 含义 |
|:---|:---|
| $X[0]$ | 直流分量（零频率） |
| $X[1] \sim X[N/2 - 1]$ | 正频率分量 |
| $X[N/2]$ | 奈奎斯特频率（最高频率） |
| $X[N/2 + 1] \sim X[N-1]$ | 负频率分量（与前一半共轭对称） |

### 1.3 $X[k + N/2]$ 的底层逻辑

旋转因子的周期性：$W_N^{k + N/2} = W_N^k \cdot W_N^{N/2} = W_N^k \cdot e^{-j\pi} = -W_N^k$

这就是 $X[k + N/2]$ 可通过 $X[k]$ 的前半部分结果快速算出的数学基础。

## 二、蝶形运算（Butterfly Operation）

### 2.1 基 2-FFT 分治策略

将长度为 $N$ 的序列按奇偶索引分解为两个 $N/2$ 子序列，递归处理：

- 偶数项：$E[k] = \text{DFT}(x[0], x[2], \dots, x[N-2])$
- 奇数项：$O[k] = \text{DFT}(x[1], x[3], \dots, x[N-1])$

### 2.2 蝶形运算公式

合并子结果时，通过蝶形运算生成完整 DFT：

$$
X[k] = E[k] + W_N^k \cdot O[k]
$$

$$
X[k + N/2] = E[k] - W_N^k \cdot O[k]
$$

每个蝶形运算：一次复数乘法（$W_N^k \cdot O[k]$）+ 两次加减 → 产生两个输出。

### 2.3 计算复杂度

| 方法 | 复杂度 | $N=8$ 示例 |
|:---|:---|:---|
| 直接 DFT | $O(N^2)$ | 64 次复数乘加 |
| FFT 蝶形运算 | $O(N \log_2 N)$ | 24 次复数乘加 |

### 2.4 8 点 FFT 完整示例

假设输入 $x[n] = [1, 1, 1, 1, 1, 1, 1, 1]$（期望输出 $X[0]=8$，其余为 $0$）。

**步骤 1：码位倒序重排**

```
原始索引：0(000) 1(001) 2(010) 3(011) 4(100) 5(101) 6(110) 7(111)
位反转后：0(000) 4(100) 2(010) 6(110) 1(001) 5(101) 3(011) 7(111)
重排序列：x[0], x[4], x[2], x[6], x[1], x[5], x[3], x[7] = [1,1,1,1,1,1,1,1]
```

**步骤 2：第 1 级（跨度=1，$W_2^0 = 1$）**

```
组1: 1+1=2, 1-1=0      组3: 1+1=2, 1-1=0
组2: 1+1=2, 1-1=0      组4: 1+1=2, 1-1=0
输出：[2, 0, 2, 0, 2, 0, 2, 0]
```

**步骤 3：第 2 级（跨度=2）**

```
组1（k=0, W₈⁰=1）: 2+1·2=4, 2-1·2=0        [索引0,2]
组2（k=0, W₈⁰=1）: 0+1·0=0, 0-1·0=0        [索引4,6]
组3（k=1, W₈²=-j）: 2+(-j)·2, 2-(-j)·2     [索引1,3]→均为0
组4（k=1, W₈²=-j）: 0+(-j)·0, 0-(-j)·0     [索引5,7]→均为0
输出：[4, 0, 0, 0, 4, 0, 0, 0]
```

**步骤 4：第 3 级（跨度=4）**

```
k=0 (W₈⁰=1): 4+1·4=8, 4-1·4=0              → X[0]=8, X[4]=0
k=1 (W₈¹=(√2/2)-j(√2/2)): 均为0              → X[1]=0, X[5]=0
k=2 (W₈²=-j): 均为0                          → X[2]=0, X[6]=0
k=3 (W₈³=-j·W₈¹): 均为0                      → X[3]=0, X[7]=0
```

**最终结果**：$X = [8, 0, 0, 0, 0, 0, 0, 0]$，与理论值一致。

### 2.5 旋转因子速查（$N=8$）

| $k$ | $W_8^k$ | 直角坐标 |
|:---|:---|:---|
| 0 | $e^0$ | $1$ |
| 1 | $e^{-j\pi/4}$ | $\frac{\sqrt{2}}{2} - j\frac{\sqrt{2}}{2}$ |
| 2 | $e^{-j\pi/2}$ | $-j$ |
| 3 | $e^{-j3\pi/4}$ | $-\frac{\sqrt{2}}{2} - j\frac{\sqrt{2}}{2}$ |
| 4 | $e^{-j\pi}$ | $-1$ |

## 三、码位倒序（Bit-Reversed Order）

### 3.1 根本原因

> 码位倒序是 FFT 分治策略中**递归奇偶分解的数学必然结果**。

每次奇偶分离相当于按二进制索引**从低位到高位**逐级分组：

- 第 1 次分解：按最低位（LSB，$b_0$）分离偶数和奇数
- 第 2 次分解：按次低位（$b_1$）继续分离
- ...
- 第 $m$ 次分解后（$N = 2^m$）：索引的二进制位顺序被完全反转

### 3.2 位反转示例（$N=8$）

```
原始索引  二进制    位反转后   十进制
0         000   →   000        0
1         001   →   100        4
2         010   →   010        2
3         011   →   110        6
4         100   →   001        1
5         101   →   101        5
6         110   →   011        3
7         111   →   111        7
```

### 3.3 DIT vs DIF 中的位反转位置

| 算法 | 输入 | 输出 |
|:---|:---|:---|
| **DIT**（时域抽取） | 需位反转排列 | 自然顺序 |
| **DIF**（频域抽取） | 自然顺序 | 需位反转排列 |

### 3.4 实现

```python
def bit_reverse(n, log2N):
    rev = 0
    for _ in range(log2N):
        rev = (rev << 1) | (n & 1)
        n >>= 1
    return rev

def bit_reverse_reorder(data, N):
    log2N = int(N.bit_length() - 1)
    for n in range(N):
        r = bit_reverse(n, log2N)
        if n < r:
            data[n], data[r] = data[r], data[n]
```

> 本质：码位倒序是 FFT 用分治策略换取计算效率时，必须支付的"结构代价"——它使递归分解后的子序列在内存中连续分布，支持原位计算。

## 四、汉宁窗（Hanning Window）

### 4.1 定义

汉宁窗是一种平滑窗函数，用于减少 FFT 频谱分析中的**频谱泄漏**。

对于长度为 $N$ 的窗，离散形式：

$$
w[n] = 0.5\left(1 - \cos\frac{2\pi n}{N-1}\right), \quad n = 0, 1, \dots, N-1
$$

窗函数值范围：$0 \le w[n] \le 1$。

### 4.2 窗函数对比

| 窗类型 | 主瓣宽度 | 旁瓣衰减 | 典型应用 |
|:---|:---|:---|:---|
| **矩形窗** | 最窄 | −13 dB | 周期完整信号 |
| **汉宁窗** | 较宽 | −31 dB | 通用分析，非周期信号 |
| **海明窗** | 类似汉宁 | −42 dB | 需更高旁瓣抑制 |
| **布莱克曼窗** | 最宽 | −58 dB | 极低泄漏需求 |

### 4.3 使用步骤

1. 生成窗函数 $w[n]$
2. 对原始信号加窗：$x_{\text{windowed}}[n] = x[n] \cdot w[n]$
3. 对加窗信号执行 FFT

### 4.4 振幅校正

加窗会减少信号总能量，需进行幅度补偿。汉宁窗补偿系数约为 **2**：

$$
\text{修正幅值} = \frac{\text{测量幅值}}{\frac{1}{N}\sum w[n]} \approx \text{测量幅值} \times 2
$$

### 4.5 Python 示例

```python
import numpy as np
import matplotlib.pyplot as plt

N = 8
n = np.arange(N)
f0 = 1
x = np.cos(2 * np.pi * f0 * n / N)  # 测试信号

window = np.hanning(N)
x_windowed = x * window

X = np.fft.fft(x_windowed)
freq = np.fft.fftfreq(N, d=1/N)

plt.stem(freq, np.abs(X))
plt.title("FFT with Hanning Window (N=8)")
plt.show()
```

## 五、频谱泄漏（Spectral Leakage）

### 5.1 原因

频谱泄漏是由于**信号截断**导致的频域能量扩散现象：

- 截断操作 = 原始信号 × 矩形窗（时域乘积）
- 频域等效 = 原始频谱 ∗ 矩形窗频谱（卷积）
- 矩形窗频谱有较高的旁瓣（−13 dB），卷积后能量扩散到非目标频率

**数学表达**：

$$
X_{\text{截断}}(f) = X(f) * W_{\text{矩形窗}}(f)
$$

### 5.2 加窗如何修复

用平滑窗替代矩形窗 → 信号两端逐渐衰减至 0 → 减少时域截断的不连续性 → 窗频谱旁瓣更低 → 卷积后能量扩散显著减少。

| | 不加窗（矩形窗） | 加汉宁窗 |
|:---|:---|:---|
| 时域 | 两端陡峭截断，不连续 | 两端平滑衰减至 0 |
| 频域 | 旁瓣高（−13 dB），泄漏严重 | 旁瓣低（−31 dB），泄漏少 |
| 代价 | — | 主瓣展宽（频率分辨率下降） |

### 5.3 何时需要加窗

- 信号在截断时**非周期完整**（信号长度不是周期的整数倍）
- 未知信号周期或非平稳信号
- 需要平衡频率分辨率与泄漏抑制（振动分析、音频处理）

### 5.4 核心权衡

> 主瓣宽度（频率分辨率）⟷ 旁瓣衰减（泄漏抑制）不可兼得，需根据实际需求选择窗函数。
