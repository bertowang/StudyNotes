# 🟢 第一部分：小白入门

## 1. LTspice 是什么？

LTspice 是 Analog Devices 出品的**免费电路仿真软件**。你可以把它理解为"电路的虚拟实验台"：

- 不需要买元件、不需要焊板子
- 画好电路图，点一下运行，就能看到电压/电流波形
- 完全免费，专业工程师也在用

**适合做什么？**

| 场景 | 举例 |
|------|------|
| 验证电路设计 | 滤波器截止频率是否正确 |
| 学习电路原理 | 观察 RC 充放电过程 |
| 调试参数 | 找最优电阻值 |
| 频率响应分析 | 画波特图 |

## 4. 实战：从零画一个 RC 电路并仿真

我们将画一个最简单的 **RC 低通滤波器**，完成从画图到看波形的全流程。

### 电路目标

```
         R1 (1kΩ)
  in ────┤├──── out
                │
               C1 (1μF)
                │
               GND (0)
```

- 输入：1kHz 正弦波，幅值 1V
- 观察：out 节点的电压波形（RC 充放电过程）
- 截止频率：f = 1/(2π×R×C) = 1/(2π×1000×0.000001) ≈ **159 Hz**

---

### Step 1：新建原理图

```
菜单栏 → File → New Schematic
```

出现空白画布，准备开始画图。

---

### Step 2：放置电阻 R1

1. 按键盘 **`R`** 键（或菜单 Edit → Add → Resistor）
2. 光标变成电阻符号，**在画布中间偏左**单击放置
3. 双击刚放置的电阻，弹出属性对话框：
   - **Resistance**：填入 `1k`
   - 点击 **OK**
4. 按 `Esc` 退出放置模式

> 💡 放置时按 `Cmd+R` 可旋转方向（水平/垂直）

---

### Step 3：放置电容 C1

1. 按键盘 **`C`** 键
2. 在电阻右侧**偏下方**单击放置（电容需要竖向，一端接 out，一端接地）
3. 双击电容，设置属性：
   - **Capacitance**：填入 `1u`
   - 点击 **OK**
4. 按 `Esc` 退出

---

### Step 4：放置电压源 V1

1. 按键盘 **`V`** 键
2. 在电阻**左侧**放置电压源（竖向，正极朝上）
3. 双击电压源，弹出属性对话框：

   **这里是关键！** 选择电源类型：

   ```
   在 "Advanced" 标签页中：
   
   ① 选择波形类型：SINE（正弦波）
   
   填写参数：
   DC offset (V)：0        ← 直流偏置为 0
   Amplitude (V)：1        ← 幅值 1V
   Freq (Hz)：1k           ← 频率 1kHz
   Tdelay (s)：0           ← 延迟 0s
   Theta (1/s)：0          ← 衰减系数 0（不衰减）
   Phi (deg)：0            ← 初始相位 0°
   Ncycles：0              ← 0 表示持续输出
   ```

   点击 **OK**

---

### Step 5：放置地（GND）

1. 按键盘 **`G`** 键（或工具栏点击 GND 图标）
2. 在电压源**负极下方**放置一个 GND
3. 在电容**下端**再放置一个 GND
4. 按 `Esc` 退出

---

### Step 6：连线

1. 按键盘 **`W`** 键进入连线模式
2. 按以下顺序连线（单击起点，单击终点，自动画线）：

   ```
   连线顺序：
   ① 电压源正极 → 电阻左端（这条线就是 "in" 节点）
   ② 电阻右端 → 电容上端（这条线就是 "out" 节点）
   ③ 电压源负极 → GND（已放置的地符号）
   ④ 电容下端 → GND（已放置的地符号）
   ```

3. 按 `Esc` 退出连线模式

---

### Step 7：添加节点标签（可选但推荐）

给关键节点命名，方便后续查看波形：

1. 菜单 **Edit → Add → Label**（或按 `L`）
2. 在电压源正极那条线上点击，输入标签名 `in`，点 OK
3. 在电阻右端那条线上点击，输入标签名 `out`，点 OK

---

### Step 8：添加仿真命令

1. 菜单 **Simulate → Edit Simulation Cmd**
2. 弹出仿真设置对话框，选择 **Transient**（瞬态分析）标签页：

   ```
   Stop Time：10m          ← 仿真 10 毫秒
   Time To Start Saving Data：0
   Maximum Timestep：1u    ← 最大步长 1 微秒（可选，留空也行）
   ```

3. 点击 **OK**，光标变成仿真命令文本
4. 在画布**空白处**单击放置（会显示 `.tran 10m` 文字）

---

### Step 9：运行仿真

点击工具栏的 **▶ 运行按钮**（或按 `Cmd+R`）

- 如果电路正确：仿真立即完成，波形窗口自动弹出（或在原理图下方出现）
- 如果报错：检查是否所有元件都接了 GND

---

### Step 10：查看波形

仿真完成后，回到原理图窗口：

1. **查看 out 节点电压**：
   - 将鼠标移到 `out` 节点的连线上
   - 光标变成**红色电压探针**（像一个小电压表）
   - 单击，波形窗口出现 `V(out)` 的波形

2. **同时查看 in 节点**：
   - 同样点击 `in` 节点的连线
   - 波形窗口叠加显示 `V(in)` 的波形

3. **你应该看到**：
   ```
   V(in)：标准正弦波，幅值 1V，频率 1kHz
   V(out)：幅值明显衰减（约 0.16V），相位滞后
   
   原因：1kHz >> 截止频率 159Hz，所以信号被大幅衰减
   ```

---

### Step 11：改变电源类型再仿真

#### 改为脉冲源（PULSE），观察 RC 充放电

双击电压源 V1，在 Advanced 标签页选择 **PULSE**：

```
Initial Value (V)：0       ← 初始电平 0V
Pulsed Value (V)：5        ← 高电平 5V
Delay Time (s)：0          ← 延迟 0s
Rise Time (s)：1n          ← 上升时间 1ns（近似阶跃）
Fall Time (s)：1n          ← 下降时间 1ns
Pulse Width (s)：5m        ← 脉宽 5ms
Period (s)：10m            ← 周期 10ms
```

重新运行仿真（`Cmd+R`），观察 `V(out)` 的指数充放电曲线：

```
时间常数 τ = R×C = 1kΩ × 1μF = 1ms
充电到 63.2% 需要 1τ = 1ms
充电到 99.3% 需要 5τ = 5ms
```

#### 改为 AC 源，观察频率响应

1. 双击电压源，选择 **SINE**，并在 **Small Signal AC Analysis** 栏填入：
   - **AC Amplitude**：`1`

2. 删除原来的 `.tran` 命令（点击文字后按 `Del`）

3. 重新添加仿真命令：**Simulate → Edit Simulation Cmd → AC Analysis**：
   ```
   Type of Sweep：Decade（对数扫频）
   Number of points per decade：100
   Start Frequency：1
   Stop Frequency：100k
   ```

4. 运行后点击 `out` 节点，看到**波特图**（幅频 + 相频曲线）

### 5.2 波形窗口基本操作

#### 查看信号（最重要的第一步）

```
方法 1：点击原理图中的导线
  → 鼠标移到原理图，光标变成"电压探针"图标（红色探针）
  → 点击任意导线 → 该节点的电压波形自动出现在波形窗口
  → 点击元件引脚 → 显示流过该元件的电流

方法 2：在波形窗口手动添加
  → 右键点击波形区空白处 → 选择 "Add Trace"
  → 在弹出框中输入表达式（详见 5.3 节）
  → 点击 OK

方法 3：Alt + 点击元件
  → 光标变成"功率探针"图标（温度计形状）
  → 点击元件 → 显示该元件的瞬时功率 P = V × I
```

#### 缩放与导航

| 操作 | 鼠标/键盘方法 | 说明 |
|------|-------------|------|
| **放大局部区域** | 左键框选（从左上到右下拖动） | 最常用，精确放大感兴趣的区域 |
| **缩小还原全图** | `Ctrl + A` | 自动缩放到显示所有数据 |
| **滚轮缩放** | 鼠标滚轮上下滚动 | 以鼠标位置为中心缩放 |
| **平移波形** | 右键拖动 | 上下左右移动视图 |
| **X轴单独缩放** | 在 X 轴上滚轮 | 只改变时间/频率轴范围 |
| **Y轴单独缩放** | 在 Y 轴上滚轮 | 只改变幅值轴范围 |
| **撤销缩放** | `Ctrl + Z` | 回到上一个视图状态 |

#### 管理信号显示

| 操作 | 方法 |
|------|------|
| **删除某条信号** | 右键点击图例中的信号名 → Delete Trace |
| **修改信号颜色** | 右键点击图例中的信号名 → Color → 选择颜色 |
| **修改线型/粗细** | 右键点击图例中的信号名 → Line Style |
| **隐藏/显示信号** | 右键点击图例中的信号名 → Don't Plot |

### 5.3 游标测量（精确读数的关键）

游标是波形窗口中最重要的测量工具，可以精确读取任意点的数值，以及两点之间的差值。

#### 放置游标

```
方法：在波形线上点击（不是在空白区域）
  → 第一次点击：放置游标 1（实线，显示绝对值）
  → 按住 Ctrl 再点击：放置游标 2（虚线，显示与游标1的差值）

游标显示的信息（在状态栏或弹出小窗口中）：
  Cursor 1:  时间 = 2.345ms,  V(out) = 3.142V
  Cursor 2:  时间 = 5.678ms,  V(out) = 1.234V
  Diff:      时间差 = 3.333ms, 电压差 = -1.908V
```

#### 游标的实际用途

```
测量上升时间（Rise Time）：
  → 游标1 放在波形从低电平上升到 10% 处
  → 游标2 放在波形上升到 90% 处
  → 读取 Diff 中的时间差 = 上升时间

测量信号周期：
  → 游标1 放在第一个波峰
  → 游标2 放在第二个波峰
  → Diff 时间差 = 周期 T
  → 频率 f = 1/T

测量峰峰值（Vpp）：
  → 游标1 放在波形最高点
  → 游标2 放在波形最低点
  → Diff 电压差的绝对值 = 峰峰值

测量相位差（两个信号之间）：
  → 先测量信号周期 T
  → 游标1 放在信号A的过零点（上升沿）
  → 游标2 放在信号B的对应过零点
  → 相位差（度）= (时间差 / T) × 360°

测量截止频率（AC 分析）：
  → 游标1 放在低频段的幅值（应接近 0dB）
  → 游标2 沿频率轴移动，找到幅值下降 3dB 的点
  → 游标2 所在的频率 = 截止频率 fc
```

#### 读取精确数值的技巧

```
技巧 1：让游标"吸附"到极值点
  → 将游标拖到波峰/波谷附近
  → 按键盘左右方向键 ← → 微调游标位置
  → 游标会自动吸附到局部极值点

技巧 2：直接在状态栏读数
  → 不需要游标，直接将鼠标悬停在波形线上
  → 状态栏实时显示鼠标位置对应的 (时间, 电压) 数值

技巡 3：查看统计值（平均值、有效值等）
  → 右键点击图例中的信号名 → "Attached Cursor"
  → 或使用菜单 View → SPICE Error Log 查看 .four 分析的统计结果
```

### 5.4 常用波形表达式大全

在 **Add Trace** 对话框中输入以下表达式，可以直接计算并显示派生量：

#### 基础量

```
V(out)              → out 节点相对地（GND）的电压
V(in, out)          → in 节点和 out 节点之间的电压差（差分电压）
I(R1)               → 流过电阻 R1 的电流（方向：从正极到负极）
I(V1)               → 流过电压源 V1 的电流
                       注意：负值表示电源在"输出"电流（正常供电状态）
I(C1)               → 流过电容 C1 的电流（充放电电流）
I(L1)               → 流过电感 L1 的电流
```

#### 功率计算

```
V(out)*I(R1)        → R1 的瞬时功率（W）
                       正值 = R1 在消耗功率；负值 = R1 在输出功率（受控源情况）
V(vcc)*I(Vcc)       → 电源的瞬时功率（负值表示电源输出功率，取绝对值即为输出功率）
abs(V(vcc)*I(Vcc))  → 电源输出功率的绝对值
```

#### 数学运算

```
V(out)/V(in)                    → 线性增益（倍数）
20*log10(abs(V(out)/V(in)))     → 增益（dB）
                                   0dB=1倍, 20dB=10倍, -20dB=0.1倍
20*log10(abs(V(out)))           → 输出电压的 dB 值（参考 1V）
V(out)-V(in)                    → 输出与输入的电压差（失调电压）
abs(V(out))                     → 电压绝对值（全波整流效果）
```

#### 微积分运算

```
ddt(V(out))         → 电压对时间的导数 dV/dt（V/s）
                       用途：查看电压变化速率，分析上升/下降斜率
ddt(I(L1))          → 电感电流的变化率 dI/dt（A/s）
                       用途：计算电感两端电压（V = L × dI/dt）
idt(I(C1))          → 电流对时间的积分（库仑，C）
                       用途：计算电容的累积电荷量
```

#### 逻辑与条件

```
V(out)>2.5          → 输出是否高于 2.5V（结果为 0 或 1，用于数字信号分析）
if(V(out)>2.5, 1, 0)→ 条件判断：高于 2.5V 显示 1，否则显示 0
```

#### 多信号组合

```
V(out)+V(in)        → 两信号叠加
V(out)-2.5          → 去除直流偏置（以 2.5V 为基准）
V(outp)-V(outn)     → 差分输出（差分放大器的真实输出）
(V(outp)-V(outn))/2 → 差分输出的半值
```

> **小技巧**：在 Add Trace 对话框中，左侧列表会列出所有可用的节点名和元件名，双击即可插入到表达式中，不需要手动输入。

### 5.5 读懂瞬态分析（.tran）结果

瞬态分析的横轴是**时间**，纵轴是**电压或电流**，就像示波器的屏幕。

#### 关键参数的读取方法

```
┌─────────────────────────────────────────────────────────┐
│  如何从波形中读取常用参数：                               │
│                                                         │
│  参数名        读取方法                                  │
│  ─────────────────────────────────────────────────────  │
│  峰值电压      游标放到波形最高点，读 Y 轴数值            │
│  峰峰值(Vpp)   游标1放最高点，游标2放最低点，读差值       │
│  有效值(RMS)   Vpp / (2√2) ≈ Vpp × 0.354（纯正弦波）    │
│  直流偏置      观察波形的"中心线"位置                    │
│  周期(T)       游标测量相邻两个同相位点的时间差           │
│  频率(f)       f = 1/T                                  │
│  上升时间      从 10% 幅值到 90% 幅值的时间              │
│  下降时间      从 90% 幅值到 10% 幅值的时间              │
│  延迟时间      输入边沿到输出 50% 幅值的时间差            │
└─────────────────────────────────────────────────────────┘
```

#### 典型波形的判断

```
正常放大器输出（1kHz 正弦输入）：
  ✅ 正常：输出是放大的正弦波，形状与输入相同
  ❌ 异常1：输出波形顶部/底部被"削平" → 削波失真，输入幅值太大
  ❌ 异常2：输出有振荡/毛刺 → 放大器不稳定，需要补偿
  ❌ 异常3：输出是直线（0V 或某固定值）→ 偏置错误，三极管截止或饱和

RC 充放电波形：
  ✅ 正常：指数曲线，时间常数 τ = R × C
  → 充电到 63.2% 的时间 = 1τ
  → 充电到 99.3% 的时间 = 5τ（认为充满）
  → 例：R=10kΩ，C=1μF → τ = 10ms，50ms 后充满

开关电源输出波形：
  ✅ 正常：有一定纹波的直流，纹波幅值 < 输出电压的 1%
  ❌ 异常：纹波过大，或有低频振荡 → 反馈环路不稳定
```

#### 如何测量 RC 时间常数

```
步骤：
1. 仿真 RC 充电电路（阶跃输入）
2. 在波形上找到充电开始的时刻 t0，电压为 V0
3. 找到最终稳定电压 V_final
4. 计算 63.2% 处的电压：V_tau = V0 + 0.632 × (V_final - V0)
5. 游标找到 V(out) = V_tau 的时刻 t1
6. τ = t1 - t0

验证：τ 应该等于 R × C 的计算值
```

### 5.6 读懂交流分析（.ac）结果——波特图

波特图由**幅频曲线**（上图）和**相频曲线**（下图）组成，横轴是频率（对数刻度）。

#### 幅频曲线（Magnitude）详解

```
纵轴单位：dB（分贝）
计算公式：dB = 20 × log₁₀(Vout / Vin)

常用 dB 值对照表：
  dB 值    电压比    含义
  ──────────────────────────────────────────
  +40 dB   100 倍   高增益放大器
  +20 dB    10 倍   10 倍放大
   +6 dB     2 倍   2 倍放大
    0 dB     1 倍   输出 = 输入（无增益无衰减）
   -3 dB   0.707倍  截止频率处（功率下降一半）← 最重要！
   -6 dB    0.5 倍  幅值减半
  -20 dB    0.1 倍  衰减到 1/10
  -40 dB   0.01 倍  衰减到 1/100

如何找截止频率（-3dB 频率）：
  1. 观察低频段的幅值（应该是一条水平线，记为 A₀ dB）
  2. 在幅频曲线上找到幅值下降到 (A₀ - 3) dB 的点
  3. 该点对应的频率就是截止频率 fc
  4. 例：低频增益 0dB，截止频率处幅值 = -3dB
```

#### 滚降斜率的含义

```
滚降斜率（截止频率之后幅值下降的速度）：

  -20 dB/decade（每十倍频衰减 20dB）
  = -6 dB/octave（每倍频衰减 6dB）
  → 一阶系统（单个 RC 或单个极点）的特征

  -40 dB/decade（每十倍频衰减 40dB）
  = -12 dB/octave
  → 二阶系统（两个极点）的特征

  -60 dB/decade → 三阶系统，以此类推

实际读取方法：
  在截止频率之后，取两个频率点（相差 10 倍）
  读取对应的 dB 值，计算差值
  例：f=10kHz 时 -20dB，f=100kHz 时 -40dB
  → 斜率 = (-40) - (-20) = -20 dB/decade → 一阶系统
```

#### 相频曲线（Phase）详解

```
纵轴单位：度（°）
含义：输出信号相对于输入信号的相位滞后（负值）或超前（正值）

常见电路的相频特征：

一阶 RC 低通滤波器：
  低频（f << fc）：相位 ≈ 0°（输出与输入同相）
  截止频率（f = fc）：相位 = -45°
  高频（f >> fc）：相位趋近 -90°

一阶 RC 高通滤波器：
  低频（f << fc）：相位 ≈ +90°（输出超前输入）
  截止频率（f = fc）：相位 = +45°
  高频（f >> fc）：相位趋近 0°

运放反相放大器：
  低频：相位 = -180°（反相）
  高频：相位继续下降（因为运放内部有极点）

稳定性判断（反馈系统）：
  相位裕量 = 增益为 0dB 时的相位 + 180°
  增益裕量 = 相位为 -180° 时的增益（取负值）
  相位裕量 > 45° → 系统稳定（推荐 > 60°）
  相位裕量 < 0°  → 系统振荡！
```

#### 波特图读取实战示例

```
示例：RC 低通滤波器（R=1kΩ，C=159nF，fc = 1kHz）

幅频曲线读取：
  100Hz：幅值 ≈ 0dB（几乎无衰减）✓
  1kHz ：幅值 ≈ -3dB（截止频率）✓
  10kHz：幅值 ≈ -20dB（衰减到 1/10）✓
  100kHz：幅值 ≈ -40dB（衰减到 1/100）✓
  → 斜率 -20dB/decade，确认为一阶系统 ✓

相频曲线读取：
  100Hz：相位 ≈ -6°（接近 0°）✓
  1kHz ：相位 ≈ -45°（截止频率处）✓
  10kHz：相位 ≈ -84°（接近 -90°）✓
  → 符合一阶低通滤波器特征 ✓
```

### 5.7 读懂直流扫描（.dc）结果

直流扫描的横轴是**扫描变量**（电压/电流/温度），纵轴是**电压或电流**。

#### 典型曲线的判读

```
二极管 I-V 特性曲线：
  横轴：正向电压 V（0 ~ 1V）
  纵轴：正向电流 I

  读取要点：
  → 找到电流开始快速上升的"膝点"电压 ≈ 0.6~0.7V（硅二极管）
  → 在工作电流（如 10mA）处读取对应的正向压降 Vf
  → 曲线越陡 = 动态电阻越小（越接近理想二极管）

MOSFET 输出特性曲线族（Id vs Vds，Vgs 为参数）：
  横轴：漏源电压 Vds
  纵轴：漏极电流 Id
  多条曲线：每条对应一个 Vgs 值

  读取要点：
  → 线性区（左侧）：Id 随 Vds 线性增加，MOSFET 像电阻
  → 饱和区（右侧平坦部分）：Id 基本不随 Vds 变化，受 Vgs 控制
  → 夹断点（线性区和饱和区的分界）：Vds = Vgs - Vth
  → 曲线间距均匀 = 线性跨导；间距越来越大 = 平方律特性

运放传输特性曲线（Vout vs Vin）：
  横轴：输入电压 Vin
  纵轴：输出电压 Vout

  读取要点：
  → 中间线性区的斜率 = 开环增益（非常陡）
  → 上下两端的平坦区 = 输出饱和（受电源电压限制）
  → 线性区的宽度 = 输入共模范围
  → 理想运放：线性区无限窄（增益无穷大）
```

#### 判断三极管工作状态

```

  Vbe > 0（正偏），Vbc < 0（反偏）→ 放大区（正常工作状态）✓
  Vbe > 0（正偏），Vbc > 0（正偏）→ 饱和区（开关电路导通状态）
  Vbe < 0（反偏），Vbc < 0（反偏）→ 截止区（开关电路断开状态）

快速检查放大器偏置是否正确：
  ✅ Vbe ≈ 0.65~0.70V（硅 NPN 三极管）
  ✅ Vce > 0.3V（不能太小，否则进入饱和）
  ✅ Ic 在设计值附近（如 1mA）
  ❌ Vbe = 0 → 基极没有偏置，三极管截止
  ❌ Vce ≈ 0.1V → 三极管饱和，无法放大
```

### 5.9 常见"看不懂"情况的排查

#### 波形是一条直线（没有变化）

```
可能原因 1：.tran 仿真，但电压源只设置了 AC，没有设置 SINE
  → 检查电压源：V1 in 0 AC 1 ← 这在 .tran 中输出 0V！
  → 修改为：V1 in 0 SINE(0 1 1k)

可能原因 2：.ac 仿真，但电压源没有设置 AC 幅值
  → 检查电压源：V1 in 0 SINE(0 1 1k) ← 这在 .ac 中没有激励！
  → 修改为：V1 in 0 AC 1 SINE(0 1 1k)

可能原因 3：电路连接错误，信号没有传到输出节点
  → 检查是否有断路（导线没有连上）
  → 检查节点名是否拼写正确
```

#### 波形有奇怪的毛刺或振荡

```
可能原因 1：仿真步长太大，错过了快速变化
  → 在 .tran 命令中减小 Tmax 参数
  → 例：.tran 0 10m 0 1n（最大步长 1ns）

可能原因 2：电路本身不稳定（放大器振荡）
  → 这是真实的电路问题，不是仿真错误
  → 需要添加补偿电容或修改反馈网络

可能原因 3：数值收敛问题（仿真器的数学问题）
  → 菜单 Simulate → Edit Simulation Cmd → 勾选 "Set Initial Conditions"
  → 或在 .op 命令后添加 .options method=gear
```

#### 幅频曲线在高频段突然变平或上翘

```
可能原因：寄生参数的影响
  → 实际电路中，导线有电感，焊盘有电容
  → 在仿真中，如果没有建模这些寄生参数，高频结果不准确
  → 这是正常现象，说明需要在模型中加入寄生参数

可能原因 2：运放的增益带宽积（GBW）限制
  → 运放在高频时增益下降，这是真实特性
  → 检查运放模型的 GBW 参数是否正确
```

## 8. 电源类型全解析

> **核心概念**：LTspice 中的"电源"不只是供电用的，更是**激励信号的来源**。不同的仿真类型（`.tran` / `.ac` / `.dc`）需要配合不同类型的电源设置，否则仿真结果会是一条直线或全为零。
>
> **电压源 vs 电流源**：
> - **电压源（V）**：强制两端电压为指定值，内阻为零（理想）
> - **电流源（I）**：强制流过的电流为指定值，内阻为无穷大（理想）
> - 两者语法完全对称，把 `V` 换成 `I` 即可

### 电源类型速查表

| 类型 | 关键字 | 波形 | 适用仿真 | 典型用途 |
|------|--------|------|---------|---------|
| 直流源 | `DC` | 水平直线 | `.op` `.dc` `.tran`（偏置） | 电源轨、偏置电压 |
| 交流小信号源 | `AC` | 单频正弦（数学上） | `.ac` | 频率响应分析 |
| 正弦波源 | `SINE` | 正弦波 | `.tran` | 模拟信号、音频、市电 |
| 脉冲源 | `PULSE` | 方波/梯形波 | `.tran` | 数字信号、PWM、时钟 |
| 分段线性源 | `PWL` | 任意折线波形 | `.tran` | 自定义波形、实测数据回放 |
| 调频源 | `SFFM` | FM 调制波 | `.tran` | 通信电路测试 |
| 受控源 | `E/G/H/F` | 由其他量决定 | 所有 | 建模增益、跨导、跨阻 |

---

### 8.1 直流源（DC）

#### 📌 是什么？

直流源输出一个**恒定不变**的电压或电流。就像电池一样，不管时间怎么变，输出始终是固定值。

```
波形示意：
电压
 5V ─────────────────────────────
 0V ──────────────────────────────→ 时间
```

#### 语法

```spice
* 电压源
Vname  正极节点  负极节点  DC  电压值

* 电流源（电流方向：从负极节点流向正极节点，即内部方向）
Iname  正极节点  负极节点  DC  电流值
```

#### 示例与说明

```spice
* ── 最常见：给电路供电 ──────────────────────────────────
VCC  vcc  0  DC 12        ; 12V 电源轨（vcc 节点相对地为 12V）
VEE  vee  0  DC -15       ; 负电源轨（运放双电源供电）
VREF ref  0  DC 2.5       ; 2.5V 参考电压

* ── 直流偏置（让三极管工作在放大区）────────────────────
Vbias base 0 DC 0.7       ; 给基极施加 0.7V 偏置

* ── 电流源供电 ──────────────────────────────────────────
I1  0  load  DC 10m       ; 10mA 恒流源（电流从 0 流向 load）
I2  vcc  col  DC 1m       ; 1mA 集电极偏置电流

* ── 配合 .dc 做扫描（DC 值会被 .dc 命令覆盖）────────────
V1  in  0  DC 0           ; 初始值 0V，.dc 命令会从 0 扫到 5V
.dc V1 0 5 0.01
```

#### ⚠️ 注意事项

```
1. DC 源在 .ac 分析中视为"短路"（电压源）或"开路"（电流源）
   → 这是正确的！因为 AC 分析只看小信号，直流偏置不影响频率响应

2. 电流源的方向容易搞混：
   I1 A B DC 1m  → 电流在元件内部从 B 流向 A（即外部从 A 流出，流向 B）
   记忆方法：正极节点（A）是电流"流出"的那端

3. 不要把 DC 源和 SINE 源搞混：
   V1 in 0 DC 5       → 永远是 5V 直线，.tran 中看不到波形变化
   V1 in 0 SINE(0 1 1k) → 才是随时间变化的正弦波
```

### 8.2 交流小信号源（AC）

#### 📌 是什么？

AC 源是专门为 `.ac` 频率扫描分析设计的**数学激励**，它本身不是真实的时域波形，而是告诉仿真引擎：

> "在每个频率点上，假设输入是幅值为 X、相位为 Y° 的正弦波，求输出的幅值和相位。"

**关键理解**：
- AC 源在 `.tran` 仿真中**完全不起作用**（不产生任何波形）
- AC 源在 `.ac` 仿真中才有效
- 通常将 AC 幅值设为 **1**（归一化），这样输出直接就是增益值

```
AC 分析的本质：
  输入：幅值 = 1V（固定），频率从 Fstart 扫到 Fstop
  输出：V(out) 的幅值和相位随频率的变化曲线
  增益 = V(out) / V(in) = V(out) / 1 = V(out)
```

#### 语法

```spice
Vname  正极  负极  AC  幅值  [相位(度)]
```

#### 示例与说明

```spice
* ── 标准用法：幅值归一化为 1 ────────────────────────────
V1  in  0  AC 1            ; 幅值 1V，相位 0°（最常用）
* 此时波特图纵轴直接显示增益（dB），0dB = 增益为 1

* ── 指定初始相位 ─────────────────────────────────────────
V1  in  0  AC 1  45        ; 幅值 1V，相位 45°
* 用于分析相位差，或多源电路中设置相对相位

* ── 同时设置 DC 偏置（建立工作点）和 AC 激励 ────────────
V1  in  0  DC 2.5  AC 1    ; 直流偏置 2.5V，AC 幅值 1V
* DC 部分用于建立三极管/MOSFET 的工作点
* AC 部分用于小信号频率响应分析

* ── 多源电路中只有一个 AC 激励 ──────────────────────────
VCC  vcc  0  DC 12         ; 电源，AC 分析中视为短路
V1   in   0  AC 1          ; 这是唯一的 AC 激励源
* 其他 DC 源在 AC 分析中自动短路（电压源）或开路（电流源）
```

#### ⚠️ 常见误区

```
误区 1：以为 AC 源在 .tran 中也能产生正弦波
  → 错！.tran 中 AC 源输出为 0V，必须用 SINE 源

误区 2：AC 幅值设得很大（如 AC 100）
  → 没必要，AC 分析是线性的，幅值只影响纵轴刻度
  → 建议始终用 AC 1，结果最直观

误区 3：忘记设置 AC 幅值，只写了 SINE
  → .ac 仿真结果全为 0，因为没有 AC 激励
  → 解决：在电压源上同时写 SINE(...) 和 AC 1
     V1 in 0 AC 1 SINE(0 1 1k)   ← 两者可以共存
```

### 8.3 正弦波源（SINE）

#### 📌 是什么？

SINE 源在 `.tran` 仿真中产生真实的**正弦波时域信号**，是模拟电路测试中最常用的激励源。

```
波形示意（SINE(0 1 1k)，即 1V 幅值、1kHz）：

电压
+1V  ·  ·  ·  ╭─╮      ╭─╮
 0V ─────────╯  ╰──────╯  ╰──→ 时间
-1V           ╰─╯      ╰─╯
              ←1ms→（一个周期）
```

#### 语法

```spice
Vname  正极  负极  SINE(Voffset  Vampl  Freq  [Tdelay  [Theta  [Phi  [Ncycles]]]])
```

#### 参数详解

| 参数 | 含义 | 单位 | 默认值 | 说明 |
|------|------|------|--------|------|
| `Voffset` | 直流偏置 | V | 0 | 正弦波的中心电平 |
| `Vampl` | 幅值（峰值） | V | — | **注意是峰值，不是有效值（RMS）** |
| `Freq` | 频率 | Hz | — | 信号频率 |
| `Tdelay` | 延迟时间 | s | 0 | 延迟期间输出 Voffset |
| `Theta` | 衰减系数 | 1/s | 0 | 0 = 不衰减；>0 = 指数衰减 |
| `Phi` | 初始相位 | 度 | 0 | 0° = 从零点开始上升 |
| `Ncycles` | 周期数 | — | 0 | 0 = 持续输出；N = 输出 N 个周期后停止 |

> **峰值 vs 有效值（RMS）**：
> - 峰值（Vampl）= 波形的最高点
> - 有效值（RMS）= 峰值 / √2 ≈ 峰值 × 0.707
> - 例：市电 220V（有效值）→ 峰值 = 220 × √2 ≈ 311V
> - 在 LTspice 中填写的是**峰值**

#### 示例与说明

```spice
* ── 最简用法：纯正弦波 ──────────────────────────────────
V1  in  0  SINE(0 1 1k)
* 含义：偏置 0V，幅值 1V（峰值），频率 1kHz
* 波形：以 0V 为中心，上下各摆动 1V，周期 1ms

* ── 带直流偏置（模拟叠加了直流的信号）────────────────────
V1  in  0  SINE(2.5 0.5 1k)
* 含义：偏置 2.5V，幅值 0.5V，频率 1kHz
* 波形：在 2V ~ 3V 之间振荡（2.5 ± 0.5）
* 用途：模拟单电源运放的输入信号（需要偏置到中间电平）

* ── 模拟市电（50Hz，220V 有效值）────────────────────────
V1  in  0  SINE(0 311 50)
* 含义：偏置 0V，峰值 311V（= 220V × √2），频率 50Hz
* 用途：测试整流电路、电源滤波器

* ── 延迟启动（模拟信号在某时刻才到来）──────────────────
V1  in  0  SINE(0 1 1k  5m)
* 含义：前 5ms 输出 0V，5ms 后开始正弦波
* 用途：测试电路的启动响应

* ── 指数衰减正弦（模拟振铃/阻尼振荡）──────────────────
V1  in  0  SINE(0 1 1k  0  500)
* 含义：正弦波幅值按 e^(-500t) 指数衰减
* 衰减时间常数 τ = 1/500 = 2ms
* 用途：模拟机械冲击响应、RLC 阻尼振荡

* ── 指定初始相位（多源相位差测试）──────────────────────
V1  in1  0  SINE(0 1 1k  0  0  0)    ; 相位 0°
V2  in2  0  SINE(0 1 1k  0  0  90)   ; 相位 90°（超前 V1 四分之一周期）
* 用途：测试移相电路、正交信号处理

* ── 限定周期数（只输出几个周期）────────────────────────
V1  in  0  SINE(0 1 1k  0  0  0  3)
* 含义：只输出 3 个完整周期，之后输出 Voffset（0V）
* 用途：模拟突发信号、脉冲群测试

* ── 同时设置 AC 幅值（.tran 和 .ac 都能用）────────────
V1  in  0  AC 1  SINE(0 1 1k)
* .tran 仿真时：产生 1kHz 正弦波
* .ac  仿真时：作为频率扫描的激励源（幅值 1V）
```

### 8.4 脉冲源（PULSE）

#### 📌 是什么？

PULSE 源产生**周期性的矩形脉冲波**（方波），是数字电路、开关电源、PWM 控制中最常用的激励源。

```
波形示意（PULSE(0 5 0 1n 1n 5u 10u)）：

电压
 5V      ┌────────┐          ┌────────┐
         │        │          │        │
 0V ─────┘        └──────────┘        └──→ 时间
         ←  5μs  →←   5μs  →
         ←────── 10μs（周期）────────→
         ↑上升沿(1ns)  ↑下降沿(1ns)
```

#### 语法

```spice
Vname  正极  负极  PULSE(Vinitial  Vpulsed  Tdelay  Trise  Tfall  Ton  Tperiod  [Ncycles])
```

#### 参数详解

| 参数 | 含义 | 单位 | 说明 |
|------|------|------|------|
| `Vinitial` | 初始（低）电平 | V | 脉冲到来前的电平，通常为 0 |
| `Vpulsed` | 脉冲（高）电平 | V | 脉冲期间的电平 |
| `Tdelay` | 延迟时间 | s | 第一个脉冲到来前的等待时间 |
| `Trise` | 上升时间 | s | 从低电平上升到高电平的时间（不能为 0！） |
| `Tfall` | 下降时间 | s | 从高电平下降到低电平的时间（不能为 0！） |
| `Ton` | 高电平持续时间 | s | 脉冲宽度（不含上升/下降时间） |
| `Tperiod` | 周期 | s | 整个脉冲的重复周期 |
| `Ncycles` | 脉冲个数 | — | 0 或省略 = 持续输出 |

> **占空比（Duty Cycle）** = Ton / Tperiod × 100%
> - 50% 占空比（方波）：Ton = Tperiod / 2
> - 25% 占空比：Ton = Tperiod / 4

> ⚠️ **Trise 和 Tfall 不能设为 0**，否则仿真会报错或不收敛。实际中设为 1ns（`1n`）即可模拟理想边沿。

#### 示例与说明

```spice
* ── 标准方波（50% 占空比）────────────────────────────────
V1  in  0  PULSE(0 5 0 1n 1n 5u 10u)
* 0→5V，上升/下降时间 1ns，高电平 5μs，周期 10μs（100kHz）
* 占空比 = 5μs / 10μs = 50%

* ── 3.3V 逻辑电平时钟信号 ───────────────────────────────
VCLK  clk  0  PULSE(0 3.3 0 1n 1n 50n 100n)
* 频率 = 1/100ns = 10MHz，3.3V 逻辑电平

* ── 延迟启动（模拟复位后才开始工作）────────────────────
V1  in  0  PULSE(0 5 1m 1n 1n 5u 10u)
* 前 1ms 保持低电平，1ms 后开始输出脉冲
* 用途：模拟上电复位时序

* ── 单次脉冲（只触发一次）──────────────────────────────
V1  in  0  PULSE(0 5 1m 1n 1n 100u 1)
* Tperiod=1（远大于仿真时间），相当于只有一个脉冲
* 用途：模拟单次触发、ESD 脉冲测试

* ── PWM 信号（可变占空比）──────────────────────────────
.param duty=0.3                        ; 占空比 30%
.param period=10u                      ; 周期 10μs
V1  in  0  PULSE(0 5 0 1n 1n {duty*period} {period})
* 用途：开关电源、电机驱动 PWM 控制

* ── 负脉冲（高电平变低电平）────────────────────────────
V1  in  0  PULSE(5 0 0 1n 1n 5u 10u)
* 初始 5V，脉冲期间降到 0V
* 用途：低电平有效的复位信号、片选信号

* ── 模拟阶跃响应（只有一个上升沿）──────────────────────
V1  in  0  PULSE(0 5 1u 1n 1n 1 2)
* 1μs 后从 0V 跳变到 5V，之后保持高电平
* 用途：测试电路对阶跃输入的响应（上升时间、过冲、稳定时间）
```

### 8.5 分段线性源（PWL）

#### 📌 是什么？

PWL（Piecewise Linear，分段线性）源通过一系列**时间-电压坐标点**来定义任意波形，相邻两点之间用直线连接。

```
波形示意（PWL(0 0  1u 0  2u 5  8u 5  9u 0  10u 0)）：

电压
 5V          ╱────────╲
            ╱          ╲
 0V ───────╱            ╲──────→ 时间
       0  1u  2u      8u 9u 10u
           ↑上升沿      ↑下降沿
```

#### 语法

```spice
* 方式 1：直接在命令中列出坐标点
Vname  正极  负极  PWL(T1 V1  T2 V2  T3 V3  ...)

* 方式 2：从外部文件读取（适合数据量大的情况）
Vname  正极  负极  PWL file="波形数据文件.txt"
```

#### 示例与说明

```spice
* ── 梯形波（有斜边的方波）──────────────────────────────
V1  in  0  PWL(0 0  1u 0  2u 5  8u 5  9u 0  10u 0)
* 时间点：0μs→1μs→2μs→8μs→9μs→10μs
* 电压值：0V → 0V → 5V → 5V → 0V → 0V
* 效果：1μs 上升沿，保持 6μs，1μs 下降沿

* ── 三角波 ──────────────────────────────────────────────
V1  in  0  PWL(0 0  5u 1  10u 0  15u -1  20u 0)
* 0→5μs：从 0V 线性上升到 1V
* 5→10μs：从 1V 线性下降到 0V
* 10→15μs：从 0V 线性下降到 -1V
* 15→20μs：从 -1V 线性上升到 0V

* ── 模拟上电时序（电源缓慢上升）────────────────────────
VCC  vcc  0  PWL(0 0  1m 0  5m 12  6m 12)
* 0~1ms：保持 0V（上电延迟）
* 1ms~5ms：从 0V 线性上升到 12V（4ms 上升时间）
* 5ms 之后：保持 12V
* 用途：测试电路在电源缓慢上升时的行为（避免上电冲击）

* ── 模拟 ESD 脉冲（IEC 61000-4-2 标准）────────────────
V1  in  0  PWL(0 0  1n 4k  30n 2k  60n 1.5k  100n 0)
* 模拟静电放电的快速上升、缓慢下降波形
* 用途：ESD 防护电路测试

* ── 回放实测数据（从示波器导出的 CSV）──────────────────
V1  in  0  PWL file="oscilloscope_data.txt"
* 文件格式（每行一个时间-电压对，空格或制表符分隔）：
* 0.000e-6   0.012
* 0.001e-6   0.015
* 0.002e-6   0.023
* ...
* 用途：用真实测量数据驱动仿真，验证电路对实际信号的响应

* ── 重复 PWL 波形（加 repeat 关键字）────────────────────
V1  in  0  PWL REPEAT FOREVER (0 0  5u 5  10u 0) ENDREPEAT
* 将括号内的波形无限重复
* 等效于周期为 10μs 的三角波
```

#### 外部文件格式说明

```
文件 waveform.txt 的格式：
  每行：时间(s)  电压(V)
  时间必须单调递增
  支持科学计数法

示例文件内容：
  0       0
  1e-6    0
  2e-6    5
  8e-6    5
  9e-6    0
  10e-6   0
```

### 8.6 调频源（SFFM）

#### 📌 是什么？

SFFM（Single-Frequency FM）是**调频（FM）信号**源，载波频率随调制信号周期性变化。

```
波形示意（调频信号）：

电压
     ↑ 频率高（密）    ↑ 频率低（疏）    ↑ 频率高（密）
  ╭╮╭╮╭╮╭╮╭╮╭╮╭╮╭╮╭╮╭╮╭─╮  ╭─╮  ╭─╮╭╮╭╮╭╮╭╮╭╮╭╮╭╮╭╮
──╯╰╯╰╯╰╯╰╯╰╯╰╯╰╯╰╯╰╯╰╯╰╯  ╰╯  ╰╯  ╰╯╰╯╰╯╰╯╰╯╰╯╰╯╰╯──→ 时间
```

#### 语法

```spice
Vname  正极  负极  SFFM(Voffset  Vampl  Fcarrier  MDI  Fsignal)
```

#### 参数详解

| 参数 | 含义 | 说明 |
|------|------|------|
| `Voffset` | 直流偏置 | 通常为 0 |
| `Vampl` | 载波幅值 | 峰值电压 |
| `Fcarrier` | 载波频率 | 中心频率 |
| `MDI` | 调制指数 | 频率偏移量 = MDI × Fsignal |
| `Fsignal` | 调制信号频率 | 频率变化的速率 |

> **调制指数 MDI 的含义**：
> - MDI = 1：频率偏移 = 1 × Fsignal（频率在 Fcarrier ± Fsignal 之间变化）
> - MDI = 5：频率偏移 = 5 × Fsignal（频率变化范围更大）

#### 示例与说明

```spice
* ── 基础 FM 信号 ────────────────────────────────────────
V1  in  0  SFFM(0 1 100k 5 1k)
* 载波 100kHz，调制频率 1kHz，调制指数 5
* 频率在 95kHz ~ 105kHz 之间以 1kHz 的速率变化
* 用途：FM 解调电路测试、锁相环（PLL）测试

* ── 小调制指数（接近纯正弦波）──────────────────────────
V1  in  0  SFFM(0 1 1Meg 0.1 10k)
* 调制指数 0.1，频率偏移很小
* 用途：测试窄带 FM 接收机

* ── 大调制指数（宽带 FM）────────────────────────────────
V1  in  0  SFFM(0 1 100Meg 75k 15k)
* 模拟 FM 广播：载波 100MHz，最大频偏 75kHz，调制频率 15kHz
```

### 8.7 受控源（VCVS / VCCS / CCVS / CCCS）

#### 📌 是什么？

受控源的输出**由电路中其他地方的电压或电流决定**，是建立电路模型（如放大器、传感器、电机）的核心工具。

```
四种受控源：

控制量 → 输出量    元件前缀    名称
─────────────────────────────────────────────
电压   → 电压      E          电压控制电压源（VCVS）
电压   → 电流      G          电压控制电流源（VCCS）
电流   → 电压      H          电流控制电压源（CCVS）
电流   → 电流      F          电流控制电流源（CCCS）
```

#### 8.7.1 电压控制电压源（VCVS）—— E 元件

```
功能：Vout = 增益 × V(控制节点+, 控制节点-)
用途：建模电压放大器、理想运放、隔离缓冲器
```

```spice
* 语法
Ename  输出+  输出-  控制+  控制-  增益

* ── 10 倍电压放大器 ──────────────────────────────────────
E1  out  0  in  0  10
* Vout = 10 × V(in, 0) = 10 × V(in)
* 用途：快速建模一个增益为 10 的放大器，不需要画出内部电路

* ── 理想运放（增益极大）─────────────────────────────────
E_opamp  out  0  inp  inn  100k
* 增益 100000（100dB），模拟理想运放的开环增益
* 配合反馈网络可以搭建各种运放电路

* ── 差分放大器 ──────────────────────────────────────────
E1  out  0  inp  inn  100
* Vout = 100 × (V(inp) - V(inn))
* 用途：仪表放大器建模

* ── 电压跟随器（增益 = 1）───────────────────────────────
E1  out  0  in  0  1
* Vout = V(in)，输入阻抗无穷大，输出阻抗为零
* 用途：建模理想缓冲器/隔离级
```

#### 8.7.2 电压控制电流源（VCCS）—— G 元件

```
功能：Iout = 跨导 × V(控制节点+, 控制节点-)
用途：建模 MOSFET（跨导 gm）、跨导放大器（OTA）
```

```spice
* 语法
Gname  输出+  输出-  控制+  控制-  跨导(A/V)

* ── MOSFET 跨导建模 ──────────────────────────────────────
Gm1  drain  source  gate  source  0.01
* Id = 0.01 × Vgs（跨导 gm = 10mA/V = 10mS）
* 用途：简化 MOSFET 小信号模型

* ── 跨导放大器（OTA）────────────────────────────────────
G1  out  0  inp  inn  1m
* Iout = 1mA/V × (V(inp) - V(inn))
* 配合负载电阻 RL：Vout = Iout × RL = gm × RL × Vin
```

#### 8.7.3 电流控制电压源（CCVS）—— H 元件

```
功能：Vout = 跨阻 × I(感测电压源)
用途：电流-电压转换（跨阻放大器）、电流感测
```

```spice
* 语法
Hname  输出+  输出-  感测电压源名  跨阻(Ω)

* 注意：必须用一个零值电压源（Vsense）来"感测"电流
Vsense  node_a  node_b  DC 0    ; 零值电压源，不影响电路，只用于测量电流

* ── 跨阻放大器（电流转电压）────────────────────────────
Vsense  in  0  DC 0             ; 感测流过 in 节点的电流
H1  out  0  Vsense  10k         ; Vout = 10kΩ × I(Vsense)
* 用途：光电二极管电流转电压、电流检测电路

* ── 电流感测电阻建模 ─────────────────────────────────────
Vsense  shunt+  shunt-  DC 0    ; 感测分流电阻上的电流
H1  sense_out  0  Vsense  0.1   ; Vout = 0.1Ω × I（模拟 0.1Ω 分流电阻）
```

#### 8.7.4 电流控制电流源（CCCS）—— F 元件

```
功能：Iout = 电流增益 × I(感测电压源)
用途：建模三极管电流放大（β）、电流镜
```

```spice
* 语法
Fname  输出+  输出-  感测电压源名  电流增益

* ── 三极管电流放大建模 ───────────────────────────────────
Vbase_sense  base  0  DC 0      ; 感测基极电流
F1  col  emit  Vbase_sense  200 ; Ic = 200 × Ib（β = 200）
* 用途：简化三极管模型，快速验证放大电路

* ── 电流镜 ──────────────────────────────────────────────
Vref_sense  ref  0  DC 0        ; 感测参考电流
F1  out  0  Vref_sense  1       ; Iout = 1 × Iref（1:1 电流镜）
F2  out2  0  Vref_sense  5      ; Iout2 = 5 × Iref（1:5 电流镜）
```

#### 8.7.5 非线性受控源（任意函数）

LTspice 支持用**数学表达式**定义非线性受控源，这是最强大的建模工具：

```spice
* 非线性电压源（B 元件）
Bname  正极  负极  V = 表达式

* 非线性电流源（B 元件）
Bname  正极  负极  I = 表达式

* ── 平方律器件（MOSFET 简化模型）───────────────────────
B1  drain  0  I = {0.5 * 2m * (V(gate)-1)^2}
* Id = 0.5 × K × (Vgs - Vth)²，K=2mA/V²，Vth=1V

* ── 限幅放大器 ──────────────────────────────────────────
B1  out  0  V = {limit(10*V(in), -5, 5)}
* Vout = 10×Vin，但限制在 -5V ~ +5V 之间

* ── 绝对值电路 ──────────────────────────────────────────
B1  out  0  V = {abs(V(in))}

* ── 温度相关模型 ─────────────────────────────────────────
B1  out  0  V = {V(in) * (1 + 0.003 * (temp - 27))}
* 输出随温度线性变化，温度系数 0.3%/°C
```

### 8.8 电源设置综合对比

#### 不同仿真类型应该用哪种电源？

```
仿真类型    需要的电源设置
─────────────────────────────────────────────────────────────
.op         DC 源（建立直流工作点）
.dc         DC 源（被 .dc 命令扫描）
.tran       SINE / PULSE / PWL / SFFM（产生时域波形）
            + DC 源（提供偏置/电源轨）
.ac         AC 源（必须设置 AC 幅值！）
            + DC 源（建立工作点）
            注意：SINE/PULSE 在 .ac 中不起作用
.noise      同 .ac（需要 AC 源）
```

#### 一个电压源同时服务多种仿真

```spice
* 最佳实践：一个源同时设置 DC 偏置、AC 幅值和时域波形
V1  in  0  DC 0  AC 1  SINE(0 1 1k)

* 解读：
* DC 0    → .op/.dc 分析时，输入为 0V 直流
* AC 1    → .ac 分析时，作为幅值 1V 的频率扫描激励
* SINE(0 1 1k) → .tran 分析时，产生 1kHz 正弦波

* 这样写一次，三种仿真都能直接用，无需修改电路！
```

## 9. 仿真命令（点命令）完全手册

> **核心概念**：LTspice 的仿真命令以 `.`（英文点）开头，写在网表中或放置在原理图画布上，告诉仿真引擎"用什么方式分析这个电路"。一个电路可以同时放置多条仿真命令，但每次运行只执行**最近一次激活的那条**（在原理图中右键点击命令可切换激活状态）。

### 9.1 瞬态分析（.tran）

#### 📌 原理与用途

瞬态分析是**最通用的仿真方式**，它在时间轴上一步一步地求解电路方程，模拟电路从 t=0 开始随时间演变的真实过程。

**适用于任何电路**，特别是：
- RC/RL/RLC 充放电过程
- 放大器对信号的放大/失真
- 开关电源的启动和稳态波形
- 数字电路的时序和毛刺
- 振荡器的起振过程
- 电机驱动、PWM 控制

#### 语法格式

```spice
.tran [Tstep] Tstop [Tstart [Tmax]] [UIC]
```

#### 参数详解

| 参数 | 是否必填 | 说明 | 建议值 |
|------|---------|------|--------|
| `Tstep` | 可选 | **建议**输出步长（LTspice 会自动调整，不是强制步长） | 信号周期的 1/100 |
| `Tstop` | **必填** | 仿真结束时间 | 覆盖 5~10 个信号周期 |
| `Tstart` | 可选 | 开始**保存数据**的时间（之前照常仿真，只是不存） | 跳过启动瞬态时使用 |
| `Tmax` | 可选 | 最大时间步长（控制精度上限） | 信号上升时间的 1/10 |
| `UIC` | 可选 | Use Initial Conditions，使用元件上的 `IC=` 初始值 | 有预充电电容时使用 |

#### 典型用法示例

```spice
* ── 基础用法 ──────────────────────────────────────────────
.tran 10m              ; 最简写法：仿真 10ms，步长全自动

* ── 控制步长（提高精度）──────────────────────────────────
.tran 1u 10m           ; 建议步长 1μs，仿真 10ms
                       ; 适合：信号频率 ~10kHz 的电路

* ── 跳过启动瞬态（节省存储空间）────────────────────────
.tran 0 100m 50m       ; 仿真 100ms，但只保存 50ms 之后的数据
                       ; 适合：开关电源等需要等待稳态的电路

* ── 完整四参数写法 ──────────────────────────────────────
.tran 0 10m 0 100n     ; 仿真 10ms，最大步长 100ns
                       ; 适合：含快速边沿（ns 级）的数字电路

* ── 使用元件初始条件 ────────────────────────────────────
C1 out 0 10u IC=5      ; 电容预充到 5V
.tran 50m UIC          ; UIC 让仿真从 IC= 指定的状态开始
                       ; 适合：模拟电路已处于某个状态时的响应
```

#### 如何为任意电路选择参数

```
第一步：确定信号的最高频率 f_max
  → Tstep 建议 ≤ 1/(100 × f_max)
  → Tmax  建议 ≤ 1/(10  × f_max)

第二步：确定需要观察的时间长度
  → Tstop = 需要观察的周期数 × 信号周期
  → 一般观察 5~10 个周期就够了

第三步：是否有启动瞬态？
  → 开关电源、振荡器：设置 Tstart 跳过前几个周期
  → 普通放大器：Tstart = 0 即可

示例：1kHz 正弦信号通过放大器
  f_max = 1kHz → Tstep = 1u，Tmax = 100u
  观察 10 个周期 → Tstop = 10m
  → .tran 1u 10m 0 100u
```

#### 波形查看技巧

```
仿真完成后，在原理图中：
  点击导线        → 查看该节点的电压 V(node)
  点击元件引脚    → 查看流过该元件的电流 I(元件名)
  Alt + 点击元件  → 查看该元件消耗的功率

在波形窗口中：
  右键 → Add Trace → 输入表达式，例如：
    V(out) - V(in)          差分电压
    V(out) / V(in)          电压比（线性增益）
    20*log10(V(out)/V(in))  增益（dB）
    I(R1) * V(R1,R2)        R1 的功率
```

### 9.2 交流分析（.ac）

#### 📌 原理与用途

交流分析（AC Analysis）是一种**线性化小信号分析**。它不是真正地在时域仿真，而是：

1. 先计算电路的直流工作点（静态偏置）
2. 在工作点处将非线性元件（三极管、MOSFET 等）**线性化**为小信号等效模型
3. 在每个频率点上求解线性方程组，得到各节点的幅度和相位

**这意味着**：
- 速度极快（不需要时域积分）
- 结果是**小信号**响应，不反映大信号失真
- 电路中必须有至少一个 `AC` 激励源

**适用于**：
- 滤波器的频率响应（截止频率、带宽）
- 放大器的增益-频率曲线（波特图）
- 反馈系统的相位裕量和增益裕量（稳定性分析）
- 阻抗随频率的变化
- 电源抑制比（PSRR）

#### 语法格式

```spice
.ac [dec|oct|lin] N Fstart Fstop
```

#### 参数详解

| 参数 | 说明 | 选择建议 |
|------|------|---------|
| `dec` | 对数扫频，每十倍频取 N 个点 | **最常用**，适合宽频率范围 |
| `oct` | 倍频程扫频，每倍频取 N 个点 | 音频领域常用 |
| `lin` | 线性扫频，总共取 N 个点 | 频率范围窄时使用（如谐振峰附近） |
| `N` | 每段的点数 | dec 模式建议 20~100；lin 模式建议 500~2000 |
| `Fstart` | 起始频率（Hz） | 比关心的最低频率低一个数量级 |
| `Fstop` | 终止频率（Hz） | 比关心的最高频率高一个数量级 |

#### 典型用法示例

```spice
* ── 宽频扫描（最常用）────────────────────────────────────
.ac dec 100 1 10Meg    ; 1Hz ~ 10MHz，每十倍频 100 点
                       ; 适合：大多数模拟电路

* ── 音频频段 ────────────────────────────────────────────
.ac dec 50 20 20k      ; 20Hz ~ 20kHz，音频范围
                       ; 适合：音频放大器、音调控制电路

* ── 射频频段 ────────────────────────────────────────────
.ac dec 20 1Meg 1G     ; 1MHz ~ 1GHz
                       ; 适合：RF 放大器、天线匹配网络

* ── 谐振峰精细扫描 ──────────────────────────────────────
.ac lin 2000 1k 10k    ; 1kHz ~ 10kHz，线性 2000 点
                       ; 适合：LC 谐振电路，需要精确找到峰值频率

* ── 电源环路稳定性分析 ──────────────────────────────────
.ac dec 100 1 1Meg     ; 分析开关电源反馈环路的相位裕量
```

#### 电源设置要点

```spice
* AC 分析必须在电压源上设置 AC 幅值，否则没有激励！
V1 in 0 AC 1           ; AC 幅值 = 1（归一化，方便直接读增益 dB）

* 可以同时设置 DC 偏置（用于建立工作点）和 AC 激励
V1 in 0 DC 2.5 AC 1    ; 直流偏置 2.5V，AC 幅值 1V

* 多源电路：只有标记了 AC 的源才是激励，其他源视为短路（电压源）或开路（电流源）
VCC vcc 0 DC 12        ; 电源，AC 分析中视为短路（内阻为 0）
V1  in  0 AC 1         ; 信号源，这是 AC 激励
```

#### 读懂波特图

```
幅频曲线（Magnitude）：
  纵轴默认单位：dB（分贝）
  0 dB   = 增益为 1（输出 = 输入）
  +20 dB = 增益为 10
  -3 dB  = 增益下降到 0.707 倍 → 这就是截止频率 fc
  -20 dB/decade = 一阶系统的滚降斜率（每十倍频衰减 20dB）
  -40 dB/decade = 二阶系统的滚降斜率

相频曲线（Phase）：
  纵轴单位：度（°）
  一阶低通滤波器：fc 处相位 = -45°，高频趋近 -90°
  一阶高通滤波器：fc 处相位 = +45°，低频趋近 +90°

稳定性判断（反馈系统）：
  相位裕量 = 增益为 0dB 时的相位 + 180°
  增益裕量 = 相位为 -180° 时的增益（取负值）
  相位裕量 > 45° 且 增益裕量 > 10dB → 系统稳定
```

### 9.3 直流扫描（.dc）

#### 📌 原理与用途

直流扫描让一个或多个直流参数（电压源、电流源、温度、元件值）在指定范围内**逐步变化**，在每个值上求解直流工作点，画出直流特性曲线。

**适用于**：
- 二极管的 I-V 特性曲线
- 三极管/MOSFET 的输出特性（Ic-Vce 曲线族）
- 运放的传输特性（输出电压 vs 输入电压）
- 稳压器的调整率
- 电路在不同电源电压下的工作状态
- 温度对电路参数的影响

#### 语法格式

```spice
.dc Src1 Start1 Stop1 Inc1 [Src2 Start2 Stop2 Inc2]
```

#### 参数详解

| 参数 | 说明 |
|------|------|
| `Src1` | 扫描的源名称（电压源名、电流源名、`TEMP`、元件名） |
| `Start1` | 起始值 |
| `Stop1` | 终止值 |
| `Inc1` | 步长（正数，方向由 Start/Stop 决定） |
| `Src2...` | 可选的第二个扫描源（外层循环，生成曲线族） |

#### 典型用法示例

```spice
* ── 基础：扫描电压源 ─────────────────────────────────────
.dc V1 0 5 0.01        ; V1 从 0V 扫到 5V，步长 10mV
                       ; 适合：运放传输特性、稳压器特性

* ── 双极性扫描 ──────────────────────────────────────────
.dc V1 -15 15 0.1      ; 适合：运放输入范围测试

* ── 二极管 I-V 特性 ─────────────────────────────────────
* 电路：V1 → R1(1k) → D1 → GND
.dc V1 0 1 0.001       ; 扫描 0~1V，步长 1mV
* 查看 I(D1) vs V(anode) 即可得到 I-V 曲线

* ── MOSFET 输出特性曲线族 ───────────────────────────────
* 电路：Vds 扫描为内层，Vgs 为外层
.dc Vds 0 5 0.05 Vgs 1 4 0.5
* 内层：Vds 从 0 扫到 5V，步长 50mV
* 外层：Vgs 取 1V, 1.5V, 2V, 2.5V, 3V, 3.5V, 4V（7条曲线）
* 查看 I(Vds) 即可得到 Id-Vds 曲线族

* ── 温度扫描 ────────────────────────────────────────────
.dc TEMP -40 125 5     ; 温度从 -40°C 到 125°C，步长 5°C
                       ; 适合：分析电路的温度漂移

* ── 元件值扫描 ──────────────────────────────────────────
.dc R1 100 10k 100     ; R1 从 100Ω 扫到 10kΩ
                       ; 注意：R1 必须是网表中实际存在的元件名
```

#### 与 .step 的区别

```
.dc  → 专门用于直流扫描，横轴就是扫描变量，纵轴是电压/电流
.step → 更通用，可以配合任何仿真类型（.tran/.ac/.dc 都可以）
        用于生成"一组参数不同的仿真曲线"

实际上 .dc Src2 可以实现和 .step 类似的效果，
但 .step 更灵活，可以扫描 .param 参数。
```

### 9.4 直流工作点（.op）

#### 📌 原理与用途

`.op` 命令让 LTspice 在 t=0 时刻求解电路的**直流稳态**，即所有电容开路、所有电感短路后的电路状态。

**适用于**：
- 检查放大器的偏置是否正确（Ic、Vce、Vbe 等）
- 确认运放的输入共模电压是否在范围内
- 验证电源轨电压是否正确
- 作为 `.tran` 仿真的起始状态（默认行为）

#### 语法格式

```spice
.op
```

#### 结果查看方式

```
方法 1：菜单 View → SPICE Error Log（或 Cmd+L）
  → 打开 .log 文件，里面列出所有节点电压和支路电流

方法 2：仿真完成后，将鼠标悬停在原理图的导线上
  → 直接显示该节点的直流电压值

方法 3：将鼠标悬停在元件上
  → 显示流过该元件的直流电流
```

#### 典型输出示例

```
.op 仿真后，.log 文件中会显示：

--- Operating Point ---
V(vcc):    12          voltage
V(base):   2.156       voltage
V(col):    6.234       voltage
V(emit):   1.456       voltage
I(Rc):     1.227e-03   device_current   ← 集电极电流 1.23mA
I(V1):    -1.5e-05     device_current   ← 信号源电流（几乎为 0）
I(Vcc):   -1.35e-03    device_current   ← 电源提供的总电流

Q1:  NPN
  Vbe:  0.700          ← 基极-发射极电压（正常约 0.6~0.7V）
  Vbc: -4.078          ← 基极-集电极电压（负值 = 正向放大区）
  Ic:   1.227e-03      ← 集电极电流
  Ib:   6.135e-06      ← 基极电流
  β:    200            ← 电流增益
```

#### 与 .tran 联合使用

```spice
* 同时运行工作点分析和瞬态分析
```

### 9.5 噪声分析（.noise）

#### 📌 原理与用途

噪声分析计算电路中各个噪声源（电阻热噪声、晶体管散粒噪声等）对指定输出节点的贡献，给出**等效输入噪声**和**输出噪声谱密度**。

**适用于**：
- 低噪声放大器（LNA）设计
- 传感器信号调理电路
- 音频前置放大器
- ADC 驱动电路
- 任何对信噪比（SNR）有要求的电路

#### 语法格式

```spice
.noise V(output) Vsrc [dec|oct|lin] N Fstart Fstop
```

| 参数 | 说明 |
|------|------|
| `V(output)` | 观察噪声的输出节点（也可以是 `V(out,ref)` 差分） |
| `Vsrc` | 参考输入源名称（用于折算等效输入噪声） |
| `dec/oct/lin` | 扫频方式，同 `.ac` |
| `N` | 每段点数 |
| `Fstart/Fstop` | 频率范围 |

#### 典型用法示例

```spice
* ── 基础噪声分析 ────────────────────────────────────────
.noise V(out) V1 dec 100 1 1Meg
* 分析 out 节点的噪声，参考源为 V1
* 结果：输出噪声谱密度（V/√Hz）和等效输入噪声（V/√Hz）

* ── 差分输出噪声 ─────────────────────────────────────────
.noise V(outp, outn) V1 dec 100 10 100k
* 适合：差分放大器、仪表放大器

* ── 同时运行 AC 和噪声分析 ──────────────────────────────
.ac dec 100 1 1Meg
.noise V(out) V1 dec 100 1 1Meg
* 两条命令可以同时存在，LTspice 会分别运行
```

#### 查看噪声结果

```
仿真完成后，在波形窗口：
  右键 → Add Trace → 输入：
    V(onoise)    → 输出噪声谱密度（V/√Hz）
    V(inoise)    → 等效输入噪声谱密度（V/√Hz）

在 .log 文件中：
  Total Output Noise Voltage：总输出噪声（V rms，在指定频率范围内积分）
  Total Equivalent Input Noise Voltage：等效输入噪声（V rms）

噪声贡献分解：
  菜单 View → SPICE Error Log → 查看各元件的噪声贡献百分比
  → 找出噪声最大的元件，优先优化
```

### 9.6 传递函数（.tf）

#### 📌 原理与用途

`.tf` 在直流工作点处计算电路的**小信号传递函数**，给出：
- 直流增益（输出/输入的比值）
- 输入阻抗
- 输出阻抗

**适用于**：
- 快速验证放大器增益
- 检查输入/输出阻抗是否满足要求
- 阻抗匹配分析
- 反馈网络的直流特性

#### 语法格式

```spice
.tf V(output) Vsrc        ; 电压传递函数
.tf I(Rload) Vsrc         ; 跨导（电流/电压）
.tf V(output) Isrc        ; 跨阻（电压/电流）
```

#### 典型用法示例

```spice
* ── 放大器增益和阻抗 ────────────────────────────────────
.tf V(out) V1
* 结果（在 .log 文件中）：
*   Transfer function:  -9.98    ← 直流增益（约 -10，反相放大器）
*   Input impedance:    1.001k   ← 输入阻抗（约 1kΩ，即 Rin）
*   Output impedance:   9.98     ← 输出阻抗（约 10Ω，运放输出阻抗）

* ── 负载电流传递函数 ─────────────────────────────────────
.tf I(RL) V1
* 结果：跨导 = I(RL) / V(in)

* ── 注意：.tf 只给直流结果，频率响应要用 .ac ──────────
```

### 9.7 傅里叶分析（.four）

#### 📌 原理与用途

`.four` 对 `.tran` 仿真结果进行**离散傅里叶变换（DFT）**，分解出各次谐波的幅度和相位，计算**总谐波失真（THD）**。

**适用于**：
- 放大器的非线性失真分析
- D 类放大器的谐波成分
- 振荡器的频谱纯度
- 电源的谐波污染
- 任何需要量化失真的电路

#### 语法格式

```spice
.four Freq [N] V(node)
```

| 参数 | 说明 |
|------|------|
| `Freq` | 基波频率（Hz），必须与信号源频率一致 |
| `N` | 分析的谐波次数（默认 9 次） |
| `V(node)` | 要分析的节点（也可以是 `I(元件名)`） |

#### 典型用法示例

```spice
* ── 放大器 THD 分析 ──────────────────────────────────────
V1 in 0 SINE(0 0.1 1k)    ; 1kHz 正弦输入
.tran 0 20m 10m            ; 仿真 20ms，跳过前 10ms（等待稳态）
.four 1k V(out)            ; 分析 out 节点的 1kHz 及其谐波

* .log 文件中的结果示例：
* Fourier components of V(out):
* DC component: 0.0001V
* Harmonic  Frequency  Magnitude   Phase    Norm.Mag   Norm.Phase
*    1       1000 Hz    9.98 V      -0.1°    1.000      0°        ← 基波
*    2       2000 Hz    0.012 V    -90.2°    0.0012     -90.1°    ← 2次谐波
*    3       3000 Hz    0.003 V    179.8°    0.0003     179.9°    ← 3次谐波
* Total Harmonic Distortion: 0.124%                               ← THD

* ── 分析更多谐波次数 ────────────────────────────────────
.four 1k 20 V(out)         ; 分析到 20 次谐波

* ── 同时分析多个节点 ────────────────────────────────────
.four 1k V(out) I(R1)
```

### 9.8 灵敏度分析（.sens）

#### 📌 原理与用途

`.sens` 计算输出量对各个元件参数的**直流灵敏度**，即"某个元件变化 1% 时，输出会变化多少"。

**适用于**：
- 精密电路设计（找出最需要精密元件的位置）
- 容差分析的前期准备
- 电路调试（找出影响最大的元件）

#### 语法格式

```spice
.sens V(output)            ; 对输出电压的灵敏度
.sens I(Rload)             ; 对输出电流的灵敏度
```

#### 典型用法示例

```spice
* 分析哪个电阻对输出电压影响最大
.sens V(out)

* .log 文件中的结果：
* DC Sensitivities of V(out):
* Element   Element Value   Sensitivity   Sensitivity/Element
* R1        1000            -9.98e-3      -9.98e-6    ← R1 变化 1Ω，输出变化 -9.98mV
* R2        10000           9.98e-4       9.98e-8
* ...
* 灵敏度越大的元件，越需要使用精密电阻
```

```spice
* 对数步进

* 元件值步进
.step R1 100 10k 100             ; R1 从 100Ω 到 10kΩ，步长 100Ω

* 温度步进
.step temp -40 125 25            ; 温度从 -40°C 到 125°C，步长 25°C

* 嵌套步进（两个参数同时扫描）
.step param R_val list 1k 10k
.step param C_val list 100n 1u
```

### 10.3 自动测量（.meas）

```spice
* 测量最大值
.meas TRAN Vmax MAX V(out)

* 测量最小值
.meas TRAN Vmin MIN V(out)

* 测量平均值
.meas TRAN Vavg AVG V(out)

* 测量上升时间（从 10% 到 90%）
.meas TRAN trise TRIG V(out) VAL={0.1*Vpeak} RISE=1
+                  TARG V(out) VAL={0.9*Vpeak} RISE=1

* 测量 -3dB 截止频率（AC 分析）
.meas AC f3dB WHEN V(out)=0.707   ; 0.707 = -3dB（幅值归一化后）

* 测量相位裕量
.meas AC phase_at_0dB FIND VP(out) WHEN V(out)=1

* 测量周期
.meas TRAN period TRIG V(out) VAL=2.5 RISE=1
+                  TARG V(out) VAL=2.5 RISE=2
```

**🔍 仿真验证要点：**
```
【.tran 瞬态】
  ✅ 观察 V(in) 和 V(out)：输出幅值应比输入小（1kHz 处约 -3dB = 0.707倍）
  ✅ 观察相位：V(out) 应滞后 V(in) 约 45°（游标测量两波形过零点的时间差）
  ✅ 将 SINE 频率改为 100Hz：输出幅值接近输入（低频几乎不衰减）
  ✅ 将 SINE 频率改为 10kHz：输出幅值大幅衰减（高频强烈衰减）

【.ac 频率响应】
  ✅ 幅频曲线：在 1kHz 处幅值应为 -3dB
  ✅ 截止频率后斜率应为 -20dB/decade（每十倍频衰减 20dB）
  ✅ 相频曲线：1kHz 处相位应为 -45°，高频趋近 -90°
  ✅ 修改 R 或 C 值，验证 fc = 1/(2πRC) 公式
```

```spice
* 调整管（PNP 功率管）
Q1  out  ctrl  in  2N2907A

* 输出分压反馈
Rfb1  out  fb   5.1k
Rfb2  fb   0    5.1k         ; Vout × Rfb2/(Rfb1+Rfb2) = Vref

* 输出滤波
Cout  out  0  100u
RL    out  0  50             ; 负载（100mA）

.model 1N4733A D(BV=5.1 IBV=49m IS=1e-14 N=1)
.model 2N2907A PNP(BF=150 IS=1e-14 VAF=100 RC=0.3)

.dc Vin 7 15 0.1             ; 扫描输入电压

.end
```

**🔍 仿真验证要点：**
```
【.dc 输入电压扫描】
  ✅ 当 Vin > 约 7V：V(out) 稳定在 10.2V（Vref × (Rfb1+Rfb2)/Rfb2）
  ✅ 线路调整率 = ΔVout / ΔVin（越小越好）
  ✅ 压差（Dropout）= Vin - Vout 的最小值（LDO 的关键参数）

【负载调整率】
  改用 .dc RL 扫描（100Ω ~ 10kΩ）：
  ✅ 观察负载变化时 V(out) 的稳定性
  ✅ 负载调整率 = ΔVout / ΔIload

【修改输出电压】
  ✅ 改变 Rfb1/Rfb2 比值：Vout = Vref × (1 + Rfb1/Rfb2)
  ✅ 验证输出电压公式
```

#### A.7.1 状态变量滤波器（同时输出低通/带通/高通）

```spice
* ============================================================
* Demo: 状态变量滤波器（KHN 拓扑）
* 同时提供低通、带通、高通三种输出
* fc = 1kHz，Q = 2
* ============================================================

VCC  vcc  0  DC 15

XU1   s1  0  vcc  vee  hp   LT1001   ; 高通输出

* 第一个积分器（U2）→ 带通输出
Ri1   hp  i1n  10k

XU2   0   i1n  vcc  vee  bp  LT1001

Ri2   bp  i2n  10k

.end
```

**🔍 仿真验证要点：**

```
  ✅ V(hp)：高通响应，1kHz 处 -3dB

【三路输出的相位关系】
  ✅ 在 1kHz 处：hp 超前 bp 90°，bp 超前 lp 90°
  ✅ 三路输出相位依次相差 90°

【改变 Q 值】
  ✅ 修改 Rfb_bp：Q 增大 → 带通峰值更高更窄
  ✅ Q > 0.5 时低通出现峰值（欠阻尼）
  ✅ 理解 Q 值对三种响应的统一影响
```

### A.9 快速导入说明

#### 如何将网表导入 LTspice（macOS）

```
步骤 1：复制上方任意网表代码块中的全部内容

步骤 2：打开 LTspice 26.0.2

步骤 3：菜单 File → New Netlist
        （注意：不是 New Schematic，是 New Netlist）

步骤 4：将复制的内容粘贴到编辑器中

步骤 5：菜单 File → Save As，保存为 xxx.net 文件
        （文件名建议与网表第二行的电路名一致）

步骤 6：菜单 Simulate → Run（或按 Cmd+R）
        → 仿真自动开始，波形窗口弹出

步骤 7：在原理图/网表窗口中，点击导线查看波形
        或右键波形区 → Add Trace 手动添加信号
```
