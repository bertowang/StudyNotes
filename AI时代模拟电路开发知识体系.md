## 二、电路基础理论

> 这是模拟电路的数学地基，无论 AI 工具多强大，理解这些基础才能看懂仿真结果、判断 AI 建议是否合理。

### 2.1 基尔霍夫定律（KCL/KVL）

```
KCL（电流定律）：节点电流代数和为零
  ∑I_in = ∑I_out
  应用：节点分析法（Node Analysis）

KVL（电压定律）：回路电压代数和为零
  ∑V = 0
  应用：网孔分析法（Mesh Analysis）

实际应用示例（运放虚短虚断）：
  虚短：V+ ≈ V-（差分输入电压趋近于零）
  虚断：I+ ≈ I- ≈ 0（输入端电流趋近于零）
  → 这两条规则是分析所有运放电路的核心工具
```

### 2.2 阻抗与频率响应

```
复阻抗（Impedance）：

  电阻：Z_R = R（与频率无关）
  电容：Z_C = 1/(jωC)  → 频率越高，阻抗越低
  电感：Z_L = jωL      → 频率越高，阻抗越高

  其中：ω = 2πf（角频率），j = √(-1)

频率响应分析：
  传递函数 H(jω) = Vout(jω) / Vin(jω)
  
  幅频响应：|H(jω)| → 增益随频率的变化
  相频响应：∠H(jω) → 相位随频率的变化
  
  波特图（Bode Plot）：
  - 横轴：频率（对数坐标）
  - 纵轴：增益（dB）和相位（度）
  - 每个极点：幅频 -20dB/十倍频，相位 -45°/十倍频
  - 每个零点：幅频 +20dB/十倍频，相位 +45°/十倍频
```

### 2.3 戴维南/诺顿定理

```
戴维南定理：任何线性电路可等效为一个电压源串联一个电阻
  Vth = 开路电压
  Rth = 独立源置零后的等效电阻

诺顿定理：任何线性电路可等效为一个电流源并联一个电阻
  In  = 短路电流
  Rn  = Rth

实际应用：
  - 分析运放输出驱动能力
  - 计算阻抗匹配条件（最大功率传输：R_load = Rth）
  - 简化复杂电路的分析
```

### 2.4 噪声理论基础

噪声是模拟电路设计的核心挑战，AI 时代仍需深刻理解：

```
主要噪声类型：

  热噪声（Johnson-Nyquist Noise）
  ├── 来源：电阻中自由电子的热运动
  ├── 功率谱密度：Sv = 4kTR（V²/Hz）
  └── 特点：白噪声（频谱平坦），与频率无关

  散粒噪声（Shot Noise）
  ├── 来源：载流子越过 PN 结的随机性
  ├── 功率谱密度：Si = 2qI（A²/Hz）
  └── 特点：白噪声，与直流电流成正比

  闪烁噪声（1/f Noise，粉红噪声）
  ├── 来源：半导体表面缺陷、载流子捕获/释放
  ├── 功率谱密度：Sv ∝ 1/f
  └── 特点：低频段主导，MOSFET 比 BJT 严重

噪声指标：
  信噪比（SNR）= 10·log(P_signal / P_noise)  [dB]
  噪声系数（NF）= SNR_in / SNR_out            [dB]
  等效输入噪声（RTI）= 输出噪声 / 增益
```

### 2.5 稳定性理论

```
负反馈系统稳定性（Bode 判据）：

  开环传递函数：T(jω) = A(jω) × β(jω)
  
  相位裕量（Phase Margin, PM）：
    在增益穿越频率（|T| = 0dB）处，相位距 -180° 的余量
    PM > 45°：稳定（推荐 > 60°）
    PM < 0°：不稳定，电路振荡

  增益裕量（Gain Margin, GM）：
    在相位穿越频率（∠T = -180°）处，增益距 0dB 的余量
    GM > 10dB：稳定

  实际意义：
  - 运放补偿电容的选择
  - DC-DC 控制环路设计
  - PLL 环路滤波器设计
```

## 三、半导体器件与模型

### 3.1 二极管

#### 3.1.1 基本特性

```
肖克利方程：I = Is × (e^(V/nVt) - 1)

  Is：反向饱和电流（~10^-12 A）
  n：理想因子（1~2）
  Vt = kT/q ≈ 26mV（室温 300K）

正向导通电压：
  硅二极管：~0.6-0.7V
  肖特基二极管：~0.2-0.4V（低正向压降，高速）
  锗二极管：~0.2-0.3V（已少用）
```

#### 3.1.2 重要二极管类型

| 类型 | 特点 | 应用 |
|-----|------|------|
| 整流二极管 | 大电流，慢速 | 电源整流 |
| 肖特基二极管 | 低压降，高速 | 开关电源、RF 检波 |
| 稳压二极管（Zener） | 反向击穿稳压 | 基准电压、保护电路 |
| 变容二极管 | 结电容随电压变化 | VCO、调谐电路 |
| PIN 二极管 | 高频开关特性 | RF 开关、衰减器 |
| 发光二极管（LED） | 电致发光 | 显示、光通信 |

### 3.2 双极型晶体管（BJT）

#### 3.2.1 工作区域

```
BJT 工作区域：

  截止区：Vbe < 0.5V，IC ≈ 0（开关断开）
  放大区：Vbe ≈ 0.7V，IC = β × IB（线性放大）
  饱和区：Vce < 0.2V，IC 不再受 IB 控制（开关导通）

关键参数：
  β（hFE）：电流增益，IC/IB，典型值 50~500
  fT：特征频率，β 降为 1 时的频率
  VA：厄利电压，影响输出阻抗（VA 越大，输出阻抗越高）
```

#### 3.2.2 BJT 小信号模型（π 模型）

```
BJT 小信号 π 模型：

        B ──┬──── rπ ────┬──── C
            │            │
           Cπ          gmVbe  Cμ（跨接 B-C）
            │            │
        E ──┴────────────┴──── E

  rπ = β / gm
  gm = IC / Vt（跨导，室温下 gm = IC / 26mV）
  ro = VA / IC（输出电阻）

BJT vs MOSFET 选择：
  BJT 优势：低噪声（1/f 噪声小）、高跨导、精确电流镜
  MOSFET 优势：高输入阻抗、低功耗、易集成
```

### 3.3 MOSFET

#### 3.3.1 工作区域

```
NMOS 工作区域（以增强型为例）：

  截止区：Vgs < Vth，ID ≈ 0
  线性区（三极管区）：Vds < Vgs - Vth
    ID = μn·Cox·(W/L)·[(Vgs-Vth)·Vds - Vds²/2]
  饱和区：Vds ≥ Vgs - Vth
    ID = (1/2)·μn·Cox·(W/L)·(Vgs-Vth)²·(1 + λVds)

关键参数：
  Vth：阈值电压（典型 0.3~0.7V）
  W/L：宽长比，决定驱动能力
  μn·Cox：工艺跨导参数
  λ：沟道长度调制系数（影响输出阻抗）
```

#### 3.3.2 MOSFET 小信号模型

```
MOSFET 小信号模型：

  G ──┬──────────────────── D
      │                     │
     Cgs               gmVgs  ro
      │                     │
  S ──┴──────────────────── S

  gm = 2·ID / (Vgs - Vth) = √(2·μn·Cox·(W/L)·ID)
  ro = 1 / (λ·ID)
  
  重要：MOSFET 栅极输入阻抗极高（理想情况下无穷大）
  实际：高频时栅极电容 Cgs、Cgd 不可忽略
```

### 3.4 工艺角（Process Corner）与 PVT 分析

```
PVT 变化对电路的影响：

  P（Process，工艺偏差）：
  ├── FF（Fast-Fast）：NMOS 和 PMOS 都快
  ├── SS（Slow-Slow）：NMOS 和 PMOS 都慢
  ├── TT（Typical-Typical）：典型值
  ├── FS（Fast NMOS, Slow PMOS）
  └── SF（Slow NMOS, Fast PMOS）

  V（Voltage，电源电压）：±10% 变化
  T（Temperature，温度）：-40°C ~ 125°C（工业级）

仿真要求：
  电路在所有 PVT 角点下都必须满足规格
  蒙特卡洛仿真：统计分析良率（Yield）
  
  AI 加速：传统蒙特卡洛需要数千次仿真，
           AI 代理模型（Surrogate Model）可减少 99% 仿真次数
```

## 四、核心模拟电路模块

### 4.1 运算放大器（Op-Amp）

#### 4.1.1 理想运放 vs 实际运放

| 参数 | 理想运放 | 实际运放（典型值） |
|-----|---------|----------------|
| 开环增益 Aol | ∞ | 100dB（10^5） |
| 输入阻抗 Zin | ∞ | 10MΩ（双极型）/ 10^12Ω（FET 输入） |
| 输出阻抗 Zout | 0 | 50~100Ω |
| 带宽 GBW | ∞ | 1MHz~1GHz |
| 失调电压 Vos | 0 | 0.1mV~10mV |
| 共模抑制比 CMRR | ∞ | 80~120dB |
| 压摆率 SR | ∞ | 0.5~1000 V/μs |

#### 4.1.2 经典运放电路

```
同相放大器：
  Vout = Vin × (1 + Rf/R1)
  输入阻抗高，适合传感器接口

反相放大器：
  Vout = -Vin × (Rf/R1)
  虚地点，输入阻抗 = R1

差分放大器：
  Vout = (V+ - V-) × Rf/R1
  共模抑制，适合桥式传感器

积分器：
  Vout = -(1/RC) × ∫Vin dt
  用于波形生成、ADC（Σ-Δ 调制器）

微分器：
  Vout = -RC × dVin/dt
  注意：对噪声敏感，实际需加限流电阻

仪表放大器（INA）：
  三运放结构，高 CMRR，可调增益
  应用：心电图、应变片、热电偶
```

#### 4.1.3 运放稳定性补偿

```
运放稳定性问题（容性负载）：

  问题：容性负载与运放输出阻抗形成额外极点
        → 相位裕量减小 → 振荡

  解决方案：
  1. 串联隔离电阻（Riso）：在输出串联 10~100Ω
  2. 超前补偿：在反馈电阻并联小电容
  3. 选用专为容性负载设计的运放（如 OPA2134）

  仿真验证：
  AC 分析 → 查看开环增益的相位裕量
  瞬态分析 → 阶跃响应是否有振铃
```

### 4.2 滤波器设计

#### 4.2.1 滤波器类型

```
按频率响应分类：
  低通滤波器（LPF）：通过低频，截止高频
  高通滤波器（HPF）：通过高频，截止低频
  带通滤波器（BPF）：通过特定频段
  带阻滤波器（BSF/Notch）：截止特定频段

按逼近函数分类：

  巴特沃斯（Butterworth）：
  ├── 特点：最平坦幅频响应（无纹波）
  ├── 缺点：过渡带较宽
  └── 应用：通用滤波，对平坦度要求高

  切比雪夫（Chebyshev）：
  ├── 特点：通带有等纹波，过渡带比巴特沃斯陡
  ├── 缺点：通带内有纹波，相位非线性
  └── 应用：需要陡峭截止的场合

  贝塞尔（Bessel）：
  ├── 特点：最线性相位（恒定群延迟）
  ├── 缺点：过渡带最宽
  └── 应用：脉冲信号处理，保持波形形状

  椭圆（Elliptic/Cauer）：
  ├── 特点：通带和阻带都有纹波，过渡带最陡
  └── 应用：需要最陡截止特性
```

#### 4.2.2 Sallen-Key 二阶低通滤波器

```
Sallen-Key 拓扑（最常用的有源滤波器）：

         R1        R2
  Vin ──┤├──┬──┤├──┬── Vout（运放输出）
            │       │
           C1      C2
            │       │
           GND  运放同相输入

传递函数：
  H(s) = ωn² / (s² + (ωn/Q)·s + ωn²)

  ωn = 1/√(R1·R2·C1·C2)  （自然频率）
  Q  = √(R1·R2·C1·C2) / (C2·(R1+R2))  （品质因数）

设计步骤（以 1kHz 巴特沃斯为例）：
  1. 确定 Q = 0.707（巴特沃斯二阶）
  2. 选 C1 = C2 = 10nF
  3. 计算 R1 = R2 = 1/(2π·f·C) ≈ 15.9kΩ
  4. 用 LTspice 仿真验证
```

### 4.3 振荡器

```
振荡器类型：

  RC 振荡器（低频，< 1MHz）：
  ├── 维恩桥振荡器：f = 1/(2πRC)，THD 低
  └── 移相振荡器：三节 RC 移相 180°

  LC 振荡器（高频，1MHz~GHz）：
  ├── 科尔皮兹（Colpitts）：电容分压反馈
  ├── 哈特莱（Hartley）：电感分压反馈
  └── 克拉普（Clapp）：科尔皮兹改进，频率稳定性更好

  晶体振荡器（高精度）：
  ├── 利用石英晶体的压电效应
  ├── Q 值极高（10^4 ~ 10^6）
  └── 频率稳定度：10^-6 ~ 10^-9

  振荡条件（巴克豪森准则）：
  ├── 环路增益 |Aβ| = 1
  └── 环路相移 = 0°（或 360°）
```

### 4.4 数据转换器（ADC/DAC）

```
ADC 关键指标：

  分辨率（Resolution）：N 位 → 2^N 个量化级
  采样率（Sample Rate）：每秒采样次数（SPS）
  SNR：信噪比（理想 N 位 ADC：SNR = 6.02N + 1.76 dB）
  ENOB：有效位数（ENOB = (SNR - 1.76) / 6.02）
  SFDR：无杂散动态范围
  THD：总谐波失真
  INL/DNL：积分/微分非线性误差

ADC 架构对比：

  Flash ADC：
  ├── 速度：最快（GHz 级）
  ├── 分辨率：低（6~8 位）
  └── 功耗：高（2^N - 1 个比较器）

  SAR ADC（逐次逼近）：
  ├── 速度：中（1kSPS ~ 10MSPS）
  ├── 分辨率：中高（12~18 位）
  └── 功耗：低，适合便携设备

  Σ-Δ ADC（过采样+噪声整形）：
  ├── 速度：慢（音频频段）
  ├── 分辨率：最高（16~24 位）
  └── 应用：音频、仪器仪表、传感器

  Pipeline ADC：
  ├── 速度：快（10~500MSPS）
  ├── 分辨率：中（10~16 位）
  └── 应用：视频、通信
```

### 4.5 锁相环（PLL）

```
PLL 系统框图：

  参考时钟 Fref
      ↓
  [相位频率检测器 PFD]
      ↓
  [电荷泵 CP + 环路滤波器 LF]
      ↓
  [压控振荡器 VCO] → 输出 Fout
      ↑
  [分频器 ÷N]

  锁定条件：Fout = N × Fref

关键参数：
  锁定范围（Lock Range）：能维持锁定的频率范围
  捕获范围（Capture Range）：能从失锁状态重新锁定的范围
  相位噪声（Phase Noise）：输出时钟的频谱纯度
  锁定时间（Lock Time）：从上电到锁定的时间

环路滤波器设计：
  决定 PLL 带宽和相位裕量
  带宽越宽：锁定快，但参考杂散抑制差
  带宽越窄：参考杂散抑制好，但锁定慢
```

## 五、高频与射频电路

### 5.1 传输线理论

```
传输线特性阻抗：Z0 = √(L/C)

  常见传输线：
  同轴线：Z0 = 50Ω（射频标准）/ 75Ω（视频标准）
  微带线：Z0 由线宽、介质厚度、介电常数决定
  差分对：Z_diff = 2 × Z_single（近似）

反射系数：Γ = (ZL - Z0) / (ZL + Z0)

  ZL = Z0：Γ = 0，无反射（完美匹配）
  ZL = 0（短路）：Γ = -1，全反射
  ZL = ∞（开路）：Γ = +1，全反射

驻波比（VSWR）：VSWR = (1 + |Γ|) / (1 - |Γ|)
  VSWR = 1：完美匹配
  VSWR < 1.5：良好匹配（射频设计目标）
```

### 5.2 阻抗匹配

```
阻抗匹配方法：

  L 型匹配网络：
  ├── 两个无源元件（L+C 或 C+L）
  ├── 只能变换阻抗，不能任意选择 Q 值
  └── 适合宽带匹配

  π 型 / T 型匹配网络：
  ├── 三个无源元件
  ├── 可以独立控制 Q 值
  └── 适合需要特定带宽的场合

  Smith 圆图：
  ├── 可视化阻抗匹配工具
  ├── 圆图上每个点代表一个归一化阻抗
  └── 串联元件：沿等电阻圆移动
      并联元件：沿等电导圆移动
```

### 5.3 低噪声放大器（LNA）

```
LNA 关键指标：
  噪声系数（NF）：越低越好（< 1dB 为优秀）
  增益（Gain）：足够大以抑制后级噪声
  IIP3（三阶截点）：线性度指标
  S11（输入反射系数）：输入匹配质量

Friis 噪声公式（级联系统）：
  F_total = F1 + (F2-1)/G1 + (F3-1)/(G1·G2) + ...

  结论：第一级 LNA 的噪声系数最关键！
        第一级增益越高，后级噪声贡献越小
```

## 六、电源电路设计

### 6.1 线性稳压器（LDO）

```
LDO 工作原理：

  Vin ──[Pass Transistor（PMOS）]── Vout
                    ↑
              [误差放大器]
              ↑         ↑
           Vref      [分压反馈]
                      ↑
                     Vout

  Vout = Vref × (1 + R1/R2)

关键参数：
  压差（Dropout Voltage）：Vin - Vout 最小值（越低越好）
  PSRR（电源抑制比）：抑制输入噪声的能力（越高越好）
  静态电流（Iq）：无负载时的消耗（越低越好，用于低功耗）
  瞬态响应：负载突变时的输出电压变化

适用场景：
  ✅ 低噪声要求（音频、RF 供电）
  ✅ 输入输出压差小
  ❌ 效率低（η = Vout/Vin），不适合大压差场合
```

### 6.2 开关电源（DC-DC）

```
Buck（降压）变换器：

  Vin ──[SW]──[L]── Vout
         ↓          ↑
        [D]        [C]
         ↓          ↓
        GND        GND

  Vout = D × Vin（D = 占空比，0~1）
  效率：η = 85%~95%（远高于 LDO）

Boost（升压）变换器：
  Vout = Vin / (1 - D)

Buck-Boost（升降压）：
  Vout = -Vin × D / (1-D)（反相输出）

控制模式：
  电压模式控制（VMC）：简单，稳定性好
  峰值电流模式控制（PCMC）：快速响应，内置过流保护
  COT（恒定导通时间）：超快瞬态响应，适合 CPU 供电

环路稳定性设计：
  Type II / Type III 补偿器
  相位裕量 > 45°，增益裕量 > 10dB
  用 LTspice AC 分析验证
```

## 七、仿真引擎与 EDA 工具

> 仿真是模拟电路设计的核心验证手段，"先仿真，后焊板"是铁律。SPICE 是模拟仿真的通用语言。

### 7.1 SPICE 仿真基础

#### 7.1.1 SPICE 是什么？

```
SPICE（Simulation Program with Integrated Circuit Emphasis）

  发展历史：
  1972年：UC Berkeley 开发原始 SPICE
  现代衍生版本：
  ├── LTspice（ADI，免费，最流行）
  ├── Hspice（Synopsys，工业标准）
  ├── Spectre（Cadence，IC 设计标准）
  ├── Ngspice（开源）
  └── PSpice（Cadence，PCB 设计常用）

SPICE 仿真类型：
  .op   → 直流工作点分析（Operating Point）
  .dc   → 直流扫描分析
  .ac   → 交流小信号分析（频率响应）
  .tran → 瞬态分析（时域波形）
  .noise → 噪声分析
  .tf   → 传递函数分析
  .sens → 灵敏度分析
  .mc   → 蒙特卡洛分析（统计良率）
  .step → 参数扫描
  .temp → 温度扫描
```

#### 7.1.2 SPICE 网表格式

```spice
* 运算放大器反相放大器仿真示例
* 标题行（第一行必须是注释）

* 电源定义
Vcc  VCC  0  DC  15V
Vee  VEE  0  DC  -15V
Vin  IN   0  AC  1  SIN(0 1 1k)   ; 1V 1kHz 正弦波

* 元器件定义（格式：名称 节点+ 节点- 值）
R1   IN   N1   10k    ; 输入电阻
Rf   OUT  N1   100k   ; 反馈电阻

* 运放实例化（使用 SPICE 模型）
XU1  N1  0  VCC  VEE  OUT  LM741  ; 同相端接地

* 仿真命令
.op                          ; 直流工作点
.ac  dec  100  1  1Meg       ; AC 分析：10倍频100点，1Hz~1MHz
.tran  1u  5m                ; 瞬态：步长1μs，总时间5ms
.noise  V(OUT)  Vin  dec  100  1  1Meg  ; 噪声分析

* 包含模型库
.lib "opamp_models.lib"

* 结束
.end
```

### 7.2 LTspice 详解

LTspice 是最流行的免费 SPICE 仿真器，由 ADI（Analog Devices）提供：

#### 7.2.1 LTspice 核心功能

```
LTspice 界面组成：

  原理图编辑器（Schematic Editor）
  ├── 放置元器件（F2）
  ├── 连线（F3）
  ├── 添加标签/节点名（F4）
  └── 添加仿真指令（S 键）

  波形查看器（Waveform Viewer）
  ├── 点击节点/元件查看波形
  ├── 支持数学运算（V(out)/V(in) 查看增益）
  ├── FFT 分析（查看频谱）
  └── 测量工具（上升时间、频率、幅值）

  SPICE 错误日志（.log 文件）
  └── 查看工作点、收敛信息
```

#### 7.2.2 LTspice 实用技巧

```
参数扫描（.step 命令）：
  .step param Rval list 1k 10k 100k
  → 自动仿真三个不同电阻值，波形叠加显示

蒙特卡洛分析：
  .mc 1000 tran 1m
  → 运行 1000 次随机参数仿真，统计良率

温度扫描：
  .step temp -40 125 25
  → 从 -40°C 到 125°C，每 25°C 仿真一次

器件容差设置：
  R1 IN N1 10k tol=5%   ; 5% 容差电阻
  → 蒙特卡洛仿真时自动随机变化

波形数学运算（在波形窗口输入）：
  V(out)/V(in)          → 电压增益
  20*log10(V(out)/V(in)) → 增益（dB）
  I(R1)*V(R1)           → 电阻功耗
```

#### 7.2.3 LTspice 仿真示例：RC 低通滤波器

```spice
* RC 低通滤波器 AC 分析
* 截止频率 fc = 1/(2πRC) = 1kHz

Vin  IN  0  AC  1  ; 1V 交流激励

R1   IN  OUT  15.9k   ; R = 15.9kΩ
C1   OUT  0   10n     ; C = 10nF

* fc = 1/(2π × 15.9k × 10n) ≈ 1kHz

.ac  dec  100  10  100k   ; 10Hz ~ 100kHz，每十倍频 100 点

* 期望结果：
* f = 1kHz 时，增益 = -3dB，相位 = -45°
* f >> 1kHz 时，增益以 -20dB/十倍频 衰减

.end
```

### 7.3 Cadence Spectre / Virtuoso

工业级 IC 设计仿真平台，用于芯片级模拟设计：

```
Spectre 相比 SPICE 的优势：

  精度：更精确的器件模型（BSIM4/BSIM-CMG）
  速度：比 SPICE 快 2~5 倍
  收敛性：更好的数值算法
  功能：支持 Verilog-A 行为模型

Virtuoso 设计流程：
  原理图（Schematic）→ 仿真（Spectre）→ 版图（Layout）
  → DRC（设计规则检查）→ LVS（版图与原理图对比）
  → 寄生参数提取（PEX）→ 后仿真（Post-Layout Sim）

典型仿真套件：
  ADE L/XL/GXL：仿真环境，支持参数扫描和优化
  Virtuoso AMS：混合信号仿真（模拟 + 数字）
  Spectre RF：射频仿真（S 参数、相位噪声、谐波平衡）
```

### 7.4 Keysight ADS（Advanced Design System）

射频/微波电路设计专用 EDA 工具：

```
ADS 核心功能：

  谐波平衡（Harmonic Balance, HB）仿真：
  ├── 分析非线性电路的稳态频域响应
  ├── 计算谐波失真（THD、IIP3、P1dB）
  └── 适合：功率放大器、混频器、振荡器

  S 参数仿真：
  ├── 分析多端口网络的散射参数
  ├── 可导入/导出 Touchstone 文件（.s2p）
  └── 适合：匹配网络、滤波器、传输线

  包络仿真（Envelope Simulation）：
  ├── 分析调制信号通过非线性电路
  └── 适合：EVM、ACPR 等通信指标

  动态链路（Dynamic Link）：
  └── 与 Momentum（电磁仿真）联合仿真
```

### 7.5 ANSYS / CST（电磁仿真）

```
电磁仿真工具（EM Simulation）：

  ANSYS HFSS：
  ├── 3D 全波电磁仿真（有限元法 FEM）
  ├── 适合：天线、连接器、封装、PCB 信号完整性
  └── 输出：S 参数、电场/磁场分布、辐射方向图

  CST Studio Suite：
  ├── 时域（FDTD）+ 频域（FEM）混合求解
  ├── 适合：EMC/EMI 分析、天线设计
  └── 速度比 HFSS 快（时域法）

  Cadence Sigrity：
  ├── PCB 级信号完整性和电源完整性仿真
  └── 适合：DDR 信号、电源分配网络（PDN）

  使用场景：
  当 SPICE 仿真不够用时（高频效应、辐射、耦合）
  → 提取 S 参数 → 导入 SPICE 作为黑盒模型
```

## 八、PCB 设计与信号完整性

### 8.1 模拟电路 PCB 设计原则

```
模拟电路 PCB 布局黄金规则：

  1. 地平面（Ground Plane）
     └── 完整的地平面层，减少地阻抗，提供屏蔽

  2. 模拟地与数字地分割
     └── 单点连接（Star Ground），防止数字噪声污染模拟地

  3. 去耦电容放置
     └── 尽量靠近 IC 电源引脚，引线越短越好
     └── 多个容值并联（100nF + 10μF）

  4. 敏感信号走线
     └── 远离时钟线、开关电源走线
     └── 差分信号等长等距走线
     └── 高阻抗节点（运放同相端）走线尽量短

  5. 屏蔽与隔离
     └── 模拟信号用地线包围（Guard Ring）
     └── 高频电路用金属屏蔽罩

  6. 电源滤波
     └── 每个 IC 独立去耦
     └── 磁珠隔离不同功能模块的电源
```
